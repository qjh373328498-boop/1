# Zotero 文献管理完全指南

> 适用对象：会计学本科生、研究生  
> 最后更新：2026-04-29

---

## 一、为什么需要文献管理工具

### 1.1 常见痛点

**你有没有遇到过这些问题：**

- ❌ 下载了几十篇 PDF，找不到哪篇是哪篇
- ❌ 写论文时要找某篇文献，翻遍电脑也找不到
- ❌ 手动敲参考文献格式，花几个小时调格式
- ❌ 引用过的文献，过几周就忘记核心观点
- ❌ 论文查重时发现某段话好像看过，但找不到原文

**解决方案：Zotero + 文献阅读笔记模板**

## 二、Zotero 安装与配置

### 2.1 下载与安装（2026 最新版）

#### Step 1：下载主程序

**官网：** https://www.zotero.org/download/

点击下载按钮，选择对应版本：
- Windows → Zotero-6.x.xx.exe
- macOS → Zotero-6.x.xx.dmg
- Linux → Zotero-6.x.xx.tar.bz2

#### Step 2：安装浏览器插件

**支持的浏览器：**
- Google Chrome
- Microsoft Edge
- Firefox
- Safari

**安装方法：**
1. 打开浏览器扩展商店
2. 搜索 "Zotero Connector"
3. 点击"添加到浏览器"

#### Step 3：注册账号

**网址：** https://www.zotero.org/user/register

**账号用途：**
- 云同步（跨设备访问文献）
- 文献笔记备份
- 团队协作共享

**免费空间：** 300MB（足够存几千条文献信息）

### 2.2 首次配置

#### 偏好设置

打开 Zotero → 编辑 → 首选项

**1. 同步设置**
```
同步 → 
  ✓ 与 Zotero 服务器同步
  输入用户名和密码
  ✓ 自动同步
```

**2. 引用设置**
```
引用 → 
  样式 → 选择 "China National Standard GB/T 7714-2015"
  如果没有，点击 "Get additional styles" 搜索添加
```

**3. 常规设置**
```
常规 → 
  ✓ 启动时自动同步
  ✓ 定期备份图书馆
```

## 三、文献采集方法

### 3.1 浏览器一键保存（最常用）

#### 知网 (CNKI)

1. 打开知网，搜索文献
2. 浏览器右上角 Zotero 图标变为"纸张"形状
3. 点击图标 → 自动保存题录信息和 PDF

**注意：** 部分学校需要配置知网 Cookie 才能下载 PDF

#### Web of Science

1. 搜索文献
2. 点击 Zotero 图标
3. 选择要保存的文献（可多选）
4. 点击"OK"

#### Google Scholar

1. 搜索文献
2. 每条结果右侧出现 Zotero 图标
3. 点击即可保存

#### 出版社官网（Elsevier, Springer, Wiley 等）

直接访问文献页面，点击 Zotero 图标即可保存

### 3.2 通过 DOI 识别

**适用场景：** 已有 PDF 文件，想补充元数据

**操作步骤：**
1. 右键 PDF 文件
2. 选择"检索元数据"
3. 输入 DOI 号
4. 自动补充标题、作者、期刊等信息

**获取 DOI 的方法：**
- PDF 首页通常标注
- Google Scholar 搜索标题
- 知网文献详情页

### 3.3 手动添加

**适用场景：** 会议论文、工作报告、未正式出版的文献

**操作步骤：**
1. 点击绿色"+"号
2. 选择文献类型（期刊论文/书籍/会议论文等）
3. 右侧信息栏填写：
   - 标题
   - 作者
   - 出版物
   - 年份
   - 卷期页码
4. 粘贴 PDF（可选）

### 3.4 批量导入

**适用场景：** 从其他工具迁移/下载了文献列表

**EndNote 导入：**
1. EndNote 导出为 XML 格式
2. Zotero: 文件 → 导入 → 选择 XML 文件

**RIS 格式导入：**
1. 数据库导出为 RIS 格式
2. Zotero: 文件 → 导入 → 选择 RIS 文件

**BibTeX 导入：**
1. 导出为 .bib 文件
2. Zotero: 文件 → 导入 → 选择 .bib 文件

## 四、文献组织与管理

### 4.1 文件夹（集合）分类

#### 推荐分类体系

```
我的图书馆
├── 00-待读 (Inbox)          # 新保存的文献，还没阅读
├── 01-在读 (Reading)        # 正在阅读的文献
├── 02-已读 (Read)           # 已读完的文献
├── 03-已引用 (Cited)        # 论文中引用的文献
│
├── 论文写作
│   ├── 毕业论文
│   │   ├── 数字化转型
│   │   ├── 财务绩效
│   │   └── 内部控制
│   └── 挑战杯论文
│
├── 课程作业
│   ├── 中级财务会计
│   ├── 财务管理
│   └── 审计学
│
└── 兴趣方向
    ├── ESG 披露
    ├── 公司治理
    └── 审计质量
```

#### 创建集合

1. 左侧面板右键 → "新建集合"
2. 输入集合名称
3. 拖拽文献到集合中

**注意：** 同一篇文献可以在多个集合中（不会重复占用空间）

### 4.2 标签管理

#### 标签用途

- 标记文献状态：`#待读 `、`#在读 `、` 精读`、` 泛读`
- 标记研究方法：`#实证研究 `、`#案例研究 `、`#文献综述`
- 标记重要程度：`#重要 `、`#一般 `、`仅供参考`
- 标记核心变量：`#数字化转型 `、`#财务绩效 `、`#内部控制`

#### 添加标签

**方法一：** 右键文献 → 添加标签

**方法二：** 右侧信息栏"标签"区域直接输入

**方法三：** 快捷键 `Ctrl+Shift+A` → 输入标签名

#### 颜色标签（Zotero 6.0+）

可以给标签分配颜色，方便可视化：

- 红色：重要
- 黄色：待读
- 绿色：已读
- 蓝色：已引用

### 4.3 高级检索

#### 快速搜索

右上角搜索框 → 输入关键词

**搜索范围：**
- 全文搜索（包括 PDF 内容）
- 仅元数据（标题、作者、摘要）

#### 条件检索

`高级检索` → 添加多个条件

**示例：**
```
查找：2020 年以后发表
AND  包含关键词"数字化转型"
AND  期刊为"会计研究"或"审计研究"
AND  标签为"重要"
```

#### 保存检索

经常使用的检索条件可以保存：
- 高级检索 → 设置条件
- 点击"保存检索"
- 下次直接调用

## 五、PDF 阅读与笔记

### 5.1 PDF 阅读器使用

Zotero 内置 PDF 阅读器（无需外部打开）

#### 基本操作

**打开 PDF：** 双击文献 → 自动在阅读器中打开

#### 标注工具

| 工具 | 快捷键 | 用途 |
|------|--------|------|
| 高亮 | H | 标记重点内容 |
| 下划线 | U | 重要概念、定义 |
| 删除线 | S | 不认同或有疑问 |
| 文本框 | T | 添加批注 |
| 便利贴 | N | 页面笔记 |

#### 颜色含义（推荐）

- **黄色**：核心观点、主要发现
- **绿色**：研究方法、数据来源
- **蓝色**：重要定义、理论框架
- **粉色**：创新点、贡献
- **橙色**：局限性、未来方向

### 5.2 笔记功能

#### 创建笔记

**方法一：** 右键文献 → 添加笔记

**方法二：** 快捷键 `Ctrl+N`

#### 笔记模板

 Zotero 中内置的笔记模板：

```markdown
## 研究问题
本文要解决的核心问题是什么？

## 理论基础

## 研究方法

## 主要发现

## 创新点

## 局限性

## 对我的启发
```

#### 从 PDF 标注自动生成笔记

1. 在 PDF 中做高亮和批注
2. 右键 PDF 标注 → "从标注创建笔记"
3. 自动提取所有标注到笔记
4. 整理和补充

**优点：** 集中查看所有标注，方便写作时引用

### 5.3 文献阅读工作流

#### 三遍阅读法

**第一遍：泛读（10 分钟）**

1. 读标题、摘要、引言
2. 跳读小标题、图表
3. 读结论
4. 判断：是否需要精读？
5. 打标签：`#泛读 `或`#待精读`

**第二遍：精读（1-2 小时）**

1. 逐段阅读，高亮重点
2. 在页边写批注（疑问、思考）
3. 记录关键数据、公式
4. 填写阅读笔记模板
5. 打标签：`#精读`

**第三遍：批判性阅读（写作前）**

1. 重读笔记和高亮
2. 思考：
   - 方法是否可靠？
   - 结论是否合理？
   - 有什么局限性？
   - 我能引用什么？
3. 写总结性评价（200-300 字）

## 六、Word 引用与写作

### 6.1 安装 Word 插件

Zotero 安装时会自动安装 Word 插件

**验证是否安装成功：**

打开 Word → 查看是否有"Zotero"选项卡

**如果没有：**
1. Zotero → 编辑 → 首选项
2. 引用 → Word 插件 → "重新安装 Word 插件"

### 6.2 插入引用

#### 步骤一：定位光标

在需要插入引用的位置点击

#### 步骤二：添加/编辑引用

**方法一：** 点击"Add/Edit Citation"

**方法二：** 快捷键 `Ctrl+Alt+A`

#### 步骤三：搜索文献

输入文献标题/作者关键词 → 选择文献 → 回车

#### 步骤四：自动生成

引用自动插入，格式如：(张敏，2023)

### 6.3 插入参考文献列表

1. 光标放在文末
2. 点击"Add/Edit Bibliography"
3. 自动生成参考文献列表

**重要：** 写作过程中，参考文献列表会随引用自动更新

### 6.4 切换引用格式

**写作用 GB/T 7714，投稿需要 APA？**

1. Document Preferences
2. 选择新格式（如 APA 7th）
3. 一键切换

**支持格式：**
- GB/T 7714-2015
- APA 7th
- MLA 9th
- Chicago 17th
- 各期刊特定格式（数千种）

### 6.5 常用操作

#### 添加页码

引用特定页码：
1. 点击已插入的引用
2. 在弹出的引用框中输入页码
3. 如：(张敏，2023, p. 45)

#### 删除引用

**不要直接删除！**

1. 点击"Add/Edit Citation"
2. 在引用框中点击文献右侧的"x"
3. 回车确认

#### 更新引用

更换了文献版本或需要修改：
1. 点击"Refresh"
2. 所有引用和参考文献列表自动更新

### 7.1 Jasminum（茉莉花）

**功能：** 知网增强、中文优化

**安装方法：**
1. Zotero → 工具 → 插件
2. 点击"安装插件"
3. 选择下载的 .xpi 文件

**下载地址：** https://github.com/l0oO/jasminum/releases

**主要功能：**

- 中文文献元数据补全
- 中文姓名格式优化
- 中文期刊分级

### 7.2 Zotero PDF Translate

**功能：** PDF 内划词翻译

**安装方法：** 同上，下载 .xpi 安装

**下载地址：** https://github.com/windingwind/zotero-pdf-translate

**配置翻译引擎：**
- 有道翻译（免费，需 API Key）
- 谷歌翻译（需网络）
- 百度翻译（免费，需 API Key）
- 彩云小译

**使用：** 在 PDF 中选中文字 → 自动翻译

### 7.3 Zotero Style

**功能：** 美化和增强界面

**下载地址：** https://github.com/damujiang/zotero-style

**主要功能：**
- 标签云显示
- 阅读进度追踪
- 阅读统计
- 美化的主题

### 7.4 Better BibTeX

**适用：** 使用 LaTeX/Overleaf 写作

**功能：**
- 自动生成引用密钥（Citation Key）
- 支持 Markdown 引用
- 自动更新 .bib 文件

**配置：**
1. 工具 → Better BibTeX

## 八、云同步与协作

### 8.1 云同步配置

#### 设置同步

1. Zotero → 编辑 → 首选项 → 同步
2. 输入 Zotero 账号
3. 选择同步内容：
   - ✓ 全库数据
   - ✓ 文献附件（PDF）
   - ✓ 笔记

#### 跨设备使用

**场景：** 在实验室电脑和宿舍电脑之间同步

**步骤：**
1. 两台电脑都安装 Zotero
2. 登录同一账号
3. 开启自动同步
4. 文献、笔记、标注自动同步

**注意：** 免费账号只有 300MB 空间

**解决方案：**
- 方案一：只同步元数据，PDF 通过 WebDAV 同步
- 方案二：付费扩容（3GB/年，约$20）
- 方案三：定期清理，只保留重要文献 PDF

### 8.2 群组协作

**适用场景：** 课题组/竞赛团队共享文献

#### 创建群组

1. Zotero 官网 → 登录 → Groups
2. Create Group
3. 设置权限：
   - Open（公开）
   - Private（私密）
4. 邀请成员（邮件）

#### 群组使用

1. Zotero 客户端 → 左侧出现群组
2. 拖拽文献到群组
3. 成员可以看到并添加笔记

**用途：**
- 课题组文献共享
- 竞赛团队资料共享
- 读书会共读文献

## 九、高效使用技巧

### 9.1 快捷键速查

| 功能 | Windows | macOS |
|------|---------|-------|
| 新建集合 | Ctrl+Shift+N | Cmd+Shift+N |
| 新建笔记 | Ctrl+N | Cmd+N |
| 添加标签 | Ctrl+Shift+A | Cmd+Shift+A |
| 查找文献 | Ctrl+Shift+F | Cmd+Shift+F |
| 插入引用 (Word) | Ctrl+Alt+A | Cmd+Opt+A |
| 刷新引用 | Ctrl+Alt+R | Cmd+Opt+R |

### 9.2 RSS 订阅追踪最新研究

**适用：** 追踪核心期刊最新发表

**步骤：**
1. 找到期刊的 RSS Feed 链接
   - 如：《会计研究》期刊页的 RSS 图标
2. Zotero → 新建来源 → RSS
3. 粘贴链接
4. 新文章自动出现，点击保存

### 9.3 定期维护

#### 每周清理

- 整理 Inbox 中的新文献
- 删除重复文献（右键 → 重复项）
- 补充缺失的元数据

#### 每月备份

1. 文件 → 导出图书馆
2. 选择格式：Zotero RDF
3. 保存备份文件

### 9.4 常见问题

**Q1: 知网文献无法下载 PDF？**

A: 配置 Cookie
1. 登录知网
2. Zotero → 编辑 → 首选项 → 高级 → Cookies
3. 添加知网域名
4. 点击"获取 Cookie"

**Q2: 文献太多找不到？**

A: 使用高级检索 + 标签过滤
- 按作者、年份、关键词组合检索
- 用标签缩小范围

**Q3: Word 插件不工作？**

A: 尝试以下操作
1. Word → 文件 → 选项 → 加载项 → 确认 Zotero 已启用
2. Zotero → 首选项 → 引用 → 重新安装 Word 插件
3. 重启 Word 和 Zotero

**Q4: 参考文献格式不对？**

A: 
1. 确认选择了正确的引用样式
2. 点击"Refresh"更新
3. 检查文献元数据是否完整

## 十、进阶：构建个人知识库

### 10.1 文献笔记 → 知识卡片

**方法：**
1. 读完文献后，用自己的话总结
2. 一条核心观点 = 一张卡片
3. 卡片之间建立关联

**示例卡片：**
```
## 数字化转型的价值效应

**核心观点：** 数字化转型通过降低成本、提升效率两条路径影响企业价值

**支持文献：** 
- 张敏 (2023): 降低成本效应显著
- 李明 (2022): 运营效率提升 15-20%

**我的思考：** 
- 两种效应在不同行业可能有差异
- 制造业以成本效应为主，服务业以效率效应为主

**关联卡片：** 
- [[成本粘性]]
- [[全要素生产率]]
```

### 10.2 与 Notion/Obsidian 联动

**方案一：Zotero → Notion**

使用插件：Notero
- 自动同步文献到 Notion
- 在 Notion 中建立知识图谱

**方案二：Zotero → Obsidian**

使用插件：Obsidian Citations
- 在 Obsidian 中引用 Zotero 文献
- 双向链接文献笔记

### 10.3 写作时的文献调用

**场景：** 写文献综述

1. 用标签筛选：`#数字化转型` + `#财务绩效`
2. 切换"笔记视图"，查看所有文献笔记
3. 复制关键观点到论文草稿
4. 插入引用

**效率提升：**
- 不用重新打开 PDF
- 核心观点一清二楚
- 引用格式自动生成
