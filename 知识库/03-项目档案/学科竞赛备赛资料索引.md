### 1.2 评分规则

**总成绩计算公式：**

```
总成绩 = 软件最终成绩 - 违规扣分

软件最终成绩 = (第 1 季度综合分 + 第 2 季度综合分 + 第 3 季度综合分 + 第 4 季度综合分) ÷ 4

每季度综合表现分数 = 盈利能力 + 营运能力 + 偿债能力 + 发展能力 + 预算能力 
                    - 报税扣分 - 紧急借款扣分 - 违规扣分

违规扣分 = 违规次数 × 5 分/次
```

**五大核心能力权重：**

| 能力维度 | 权重 | 关键指标 |
|----------|------|----------|
| 盈利能力 | 30% | 净利润率、净资产收益率、总资产报酬率 |
| 营运能力 | 25% | 应收账款周转率、存货周转率、总资产周转率 |
| 偿债能力 | 20% | 流动比率、速动比率、资产负债率 |
| 发展能力 | 15% | 销售增长率、资本积累率、总资产增长率 |
| 预算能力 | 10% | 预算准确率、费用控制偏差率 |

### 2.6 参赛作品核心内容

**报告题目：** 《大学生耕地保护认知与行为意愿调查报告》

**研究框架：**
```
认知 → 态度 → 意愿 → 行为
         ↓
    调节变量：主观规范、政策信任、参与渠道
```

**样本情况：**
- 调查范围：全国 31 个省（自治区、直辖市）
- 样本量：526 名在校大学生
- 分析方法：描述性统计、探索性因子分析、Logistic 回归、调节效应检验

**核心发现：**
1. 大学生耕地保护认知平均得分 71.2 分（中等偏上）
2. 75.8% 表示愿意参与耕地保护，但知行分离明显
3. 专业背景、生源地、政策关注度显著影响认知水平
4. 主观规范在认知与意愿之间起正向调节作用

**创新点：**
- 国际比较：中美日韩德五国大学生认知对比
- 政策落地路径图：12 条具体对策建议
- 研究伦理说明：知情同意、隐私保护、数据安全
