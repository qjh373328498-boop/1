## 📝 正文结构

### 一、标题（黑体三号，居中）

#### （一）一级标题（黑体四号）

##### 1. 二级标题（楷体四号）

正文内容（宋体小四，1.5 倍行距）

---

### 摘要（可选）

> 摘要：简要概括论文的核心观点和研究结论，200-300 字。
>
> 关键词：XXX；XXX；XXX（3-5 个）

---

### 正文内容

#### 1. 引言/问题提出

- 研究背景
- 问题意义
- 研究思路

#### 2. 理论基础

- 核心概念界定
- 相关理论综述
- 分析框架构建

#### 3. 分析与论证

- 论点 1 + 论据支撑
- 论点 2 + 论据支撑
- 论点 3 + 论据支撑

#### 4. 结论与启示

- 主要结论
- 现实启示
- 研究局限

## 📚 参考文献格式

### 图书
```
[序号] 作者。书名 [M].出版地：出版社，出版年：起止页码.
示例：[1] 马克思恩格斯选集（第 1 卷）[M].北京：人民出版社，2012:45-50.
```

### 期刊论文
```
[序号] 作者。文章名 [J].期刊名，年，卷 (期):起止页码.
示例：[2] 张三。马克思主义中国化研究 [J].哲学研究，2025(3):12-18.
```

### 网络资源
```
[序号] 作者。文章名 [EB/OL].(更新日期)[引用日期].URL
示例：[3] 李四。新时代马克思主义发展 [EB/OL].(2025-03-15)[2025-04-20].http://xxx.com.
```

## 💡 广工论文评分标准参考

| 评分维度 | 占比 | 说明 |
|----------|------|------|
| 选题价值 | 15% | 理论意义或现实意义 |
| 论点明确性 | 25% | 观点清晰、逻辑严密 |
| 论证充分性 | 30% | 论据充足、分析深入 |
| 格式规范 | 15% | 符合学校论文格式要求 |
| 创新体现 | 15% | 有独立思考或新见解 |

## 📌 注意事项

1. **提交格式**：通常为 Word(.docx) + PDF 双格式
2. **提交流程**：学习通/蕴瑜课堂 → 课程作业 → 上传附件
3. **命名规范**：`学号_姓名_课程论文题目.docx`
   - 示例：`3124003794_梁著星_马克思主义基本原理 AI 应用作业.docx`
4. **AI 使用声明**：如使用 AI 辅助，建议在文末注明
