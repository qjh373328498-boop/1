## 📝 常见申请表类型

### 一、奖学金申请表

#### 填写要点

| 字段 | 填写说明 | 示例 |
|------|----------|------|
| 申请类别 | □国家奖学金 □励志奖学金 □校一等奖 | ✓ 国家奖学金 |
| 学业成绩 | 专业排名 X/XX，前 X% | 3/120，前 2.5% |
| 综测成绩 | XX 分，专业排名 X | 92.5 分，第 2 名 |
| 获奖情况 | 按级别降序排列 | 国赛→省赛→校赛 |
| 申请理由 | 200-300 字，突出优势 | 见下方模板 |

#### 申请理由模板

```
尊敬的评审委员会：

    我是 XX 学院 XX 专业 XX 班的学生 XXX，学号 XXXXXXXX。
现申请 XXXX 奖学金，理由如下：

    一、学业成绩优异
    本人 20XX-20XX 学年学业成绩排名专业第 X（X/XX），
综测成绩 XX 分，获得 XXX 等 X 项荣誉。

    二、科研能力突出
    参与国家级大创项目 X 项（排名第 X），发表学术论文 X 篇，
获 XXX 竞赛 XX 等奖。

    三、综合素质全面
    担任 XXXX 职务，组织 XXXX 活动 X 次，志愿服务 XX 小时，
获得"XXXX"称号。

    四、家庭经济情况（如适用）
    来自 XX 省 XX 市农村/城镇，家庭经济困难，已办理助学贷款
XX 元。

    恳请评审委员会审核批准。

此致
敬礼！

                                    申请人：XXX
                                    20XX 年 X 月 X 日
```

### 二、评优申请表（十佳百优）

#### 填写要点

| 评选类型 | 申报条件 | 材料要求 |
|----------|----------|----------|
| 十佳大学生 | 综合排名前 5% | 事迹材料 3000 字 |
| 优秀共青团员 | 团员，评议合格 | 申报表 + 事迹 |
| 优秀学生干部 | 任职满 1 年 | 任职证明 + 事迹 |
| 社会实践先进个人 | 参与实践项目 | 实践报告 + 证明 |

#### 事迹材料结构

```
标题：【个人事迹标题】—— XXX 同学的成长故事

【导语】（50 字）
简要介绍申请人基本情况

【正文】
一、坚定理想信念，做有志青年
   - 思想政治表现
   - 理想信念追求

二、勤奋学习钻研，做有识青年
   - 学业成绩
   - 科研竞赛

三、勇于实践锻炼，做有为青年
   - 学生工作
   - 社会实践
   - 志愿服务

四、严于律己修身，做有德青年
   - 品德修养
   - 生活作风

【结语】（50 字）
总结 + 展望
```

### 四、比赛报名表

#### 常见比赛类型

| 比赛名称 | 报名平台 | 截止时间 |
|----------|----------|----------|
| 互联网 + | 全国大学生创业服务网 | 每年 7 月 |
| 挑战杯 | 挑战杯官网 | 每年 3 月 |
| 学创杯 | 学创杯官网 | 每年 4 月 |
| 国创赛 | 广工大创系统 | 每年 5 月 |

#### 报名表填写说明

| 字段 | 填写要求 | 注意事项 |
|------|----------|----------|
| 项目名称 | ≤20 字，突出核心 | 避免夸张词汇 |
| 项目类别 | 创新/创业/创业实践 | 与内容匹配 |
| 负责人 | 1 人，本科生/研究生 | 可跨专业组队 |
| 团队成员 | ≤10 人，含指导教师 | 排序按贡献 |
| 指导教师 | ≤3 人，含 1 名负责人 | 需本人同意 |

## ✅ 通用检查清单

| 检查项 | 要求 | 完成 |
|--------|------|------|
| 基本信息 | 姓名、学号、学院无误 | □ |
| 照片 | 一寸免冠证件照（蓝底） | □ |

## 📌 广工提交要点

### 1. 线上提交

```
平台：学生管理系统/学习通/蕴瑜课堂
格式：PDF（防止格式错乱）
命名：学号_姓名_申请表类型
示例：3124003794_梁著星_国家奖学金申请表.pdf
```

### 2. 线下提交

```
提交地点：学院学工办/团委办公室
提交时间：工作日 8:30-11:30, 14:30-17:00
材料顺序：申请表 + 附件（A4 装订）
```
