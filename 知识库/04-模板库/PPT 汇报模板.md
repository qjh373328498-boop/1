## 🎨 视觉规范

### 一、配色方案

#### 广工标准色

| 颜色 | 色值 | 用途 |
|------|------|------|
| 广工红 | `#C41E3A` | 主色调、标题 |
| 深灰 | `#333333` | 正文、边框 |
| 浅灰 | `#F5F5F5` | 背景、分隔 |

#### 辅助色

| 颜色 | 色值 | 用途 |
|------|------|------|
| 蓝色 | `#0066CC` | 数据图表、链接 |
| 绿色 | `#28A745` | 成功、正面信息 |
| 橙色 | `#FF9900` | 强调、警示 |

## 📐 版式规范

### 标准页面布局

```
┌────────────────────────────────────┐
│  [页眉：广东工业大学 + 校徽]        │ ← 高度 10%
├────────────────────────────────────┤
│                                    │
│                                    │
│  [主体内容区]                       │ ← 高度 70%
│                                    │
│                                    │
├────────────────────────────────────┤
│  [页脚：页码 + 日期]                │ ← 高度 20%
└────────────────────────────────────┘
```

### 留白标准

- **页边距**：上下左右各 2.5cm
- **行间距**：1.2-1.5 倍
- **段落间距**：1 行

## 📝 内容规范

### 字数限制

| 页面类型 | 字数上限 | 说明 |
|----------|----------|------|
| 封面 | 50 字 | 标题 + 信息 |
| 目录 | 30 字 | 章节标题 |
| 内容页 | 100 字/页 | 每页不超过 5 个要点 |
| 图表页 | 50 字 | 图注 + 结论 |

### 内容原则

- **KISS 原则**：Keep It Simple, Stupid
- **一页一观点**：每页只表达一个核心观点
- **图优于表，表优于文**
- **数据可视化**：数字要用图表呈现

## ⏱️ 汇报时间分配

| 环节 | 时间 | 页数参考 |
|------|------|----------|
| 开场致辞 | 30 秒 | 1 页（封面） |
| 项目背景 | 2 分钟 | 2-3 页 |
| 核心内容 | 5 分钟 | 5-7 页 |
| 成果展示 | 3 分钟 | 3-4 页 |
| 总结展望 | 1 分钟 | 1-2 页 |
| Q&A | 2 分钟 | - |
| **合计** | **约 15 分钟** | **15-20 页** |

## 📌 广工 PPT 提交要点

1. **文件格式**：PPTX + PDF 双备份
2. **嵌入字体**：文件→选项→保存→嵌入字体
3. **版本控制**：文件名标注版本号 `v1.0`、`v2.0`
4. **备份方式**：U 盘 + 邮箱 + 网盘三重备份
