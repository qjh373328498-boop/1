## 🐛 故障排除

### 问题 1：edge-tts 未安装

```bash
⚠️  edge-tts 未安装，将跳过配音生成
```

**解决**:
```bash
pip3 install edge-tts
```

### 问题 2：图片生成失败

```bash
⚠️  图片生成失败，使用占位图片
```

**解决**:
1. 检查 API key 是否正确
2. 手动准备图片放入 `images/`
3. 使用占位图继续流程

### 问题 3：FFmpeg 合成失败

```bash
Invalid argument
```

**原因**: 图片尺寸不一致

**解决**:
```bash
cd video-output-*/
for img in images/*.png; do
  ffmpeg -y -i "$img" -vf "scale=1280:720:force_original_aspect_ratio=decrease,pad=1280:720:(ow-iw)/2:(oh-ih)/2,setsar=1" "scaled/$(basename $img)"
done
```
