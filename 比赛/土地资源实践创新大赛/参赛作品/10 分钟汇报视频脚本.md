## 制作注意事项

### 画面要求

| 项目 | 标准 |
|------|------|
| 分辨率 | 1920×1080（1080P） |
| 帧率 | 30fps |
| 格式 | MP4（H.264编码） |
| 大小 | ≤200MB |
| 时长 | 9 分 30 秒 -10 分 00 秒 |

### 配音要求

- 语速：约 180 字/分钟（正常语速）
- 语调：平稳、清晰、有感染力
- 设备：专业麦克风，避免环境噪音
- 后期：降噪处理，音量标准化

### 背景音乐

- 风格：轻柔钢琴曲/轻音乐
- 音量：-20dB 至 -25dB（不盖过配音）
- 来源：免版权音乐库（如YouTube Audio Library）

### 素材建议

| 类型 | 来源 |
|------|------|
| 政策文件 | 政府官网截图 |
| 数据图表 | Excel/SPSS导出，AI 美化 |
| 地图素材 | 自然资源部标准地图 |
| 访谈画面 | 实拍或剪影 + 引述 |
| 团队照片 | 工作照/合影 |

### 制作软件推荐

- **PPT**：制作基础幻灯片
- **剪映/PR**：视频剪辑
- **AE**：特效动画（可选）
- **AU**：音频处理
- **格式工厂**：格式转换
