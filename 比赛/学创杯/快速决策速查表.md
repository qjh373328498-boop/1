## 一、核心公式速查

### 现金预算

```
期末现金 = 期初现金 + 销售回款 + 融资流入 - 采购支出 - 费用支出 - 还本付息

可动用资金 = 期初现金 × 80%（保留 20% 安全边际）

最高贷款额度 = 可动用资金 × 2（不超过 4 倍杠杆）
```

### 报价计算

```
保本价 = 单位材料成本 + 单位人工成本 + 单位制造费用

目标价 = 保本价 × (1 + 目标利润率)

市场抢占价 = 保本价 × 0.85 ~ 0.90
利润导向价 = 保本价 × 1.05 ~ 1.10
```

### 广告 ROI

```
广告 ROI = 新增订单金额 ÷ 广告投入

合理区间：1:5 ~ 1:8

例：投入 10 万广告 → 应获得 50-80 万订单
```

## 四、快速决策矩阵

### 场景 1：现金紧张

| 情况 | 优先操作 | 备选方案 |
|------|---------|---------|
| 现金余额<10 万 | 申请短期贷款 | 延迟设备投资 |
| 现金余额<5 万 | 申请紧急贷款（扣分） | 变卖闲置资产 |
| 贷款额度已满 | 股权融资 | 减少订单量 |

### 场景 2：订单不足

| 情况 | 应对策略 |
|------|---------|
| 中标率<30% | 下调报价 5-10%，增加广告投放 |
| 中标率 30-50% | 维持报价，优化广告渠道 |
| 中标率>70% | 可上调报价 5%，提升利润 |

### 场景 3：产能过剩

| 情况 | 处理方式 |
|------|---------|
| 设备闲置>30% | 减少新设备投入，接低价订单填产能 |
| 库存积压 | 折价促销，减少下期生产 |
| 人员冗余 | 停止招聘，自然流失 |

### 场景 4：产能不足

| 情况 | 处理方式 |
|------|---------|
| 订单接不下 | 优先高利润订单，外包部分生产 |
| 交付压力大 | 增加班次，优化生产流程 |
| 设备满负荷 | 购置新设备（评估 ROI） |

## 五、财务指标计算（手算版）

### 快速计算法

```
净利润率 ≈ 净利润 ÷ 销售收入（保留 2 位小数）

例：85000 ÷ 1200000 = 0.0708 = 7.08%

流动比率 = 流动资产 ÷ 流动负债（保留 1 位小数）

例：350000 ÷ 280000 = 1.25

资产负债率 = 负债 ÷ 资产 × 100%（保留整数）

例：450000 ÷ 800000 × 100% = 56%
```

### 目标值参考

| 指标 | 优秀 | 良好 | 合格 | 危险 |
|------|------|------|------|------|
| 净利润率 | >10% | 7-10% | 5-7% | <5% |
| 流动比率 | >1.5 | 1.2-1.5 | 1.0-1.2 | <1.0 |
| 资产负债率 | <50% | 50-60% | 60-70% | >70% |
| 应收账款周转率 | >6 | 4-6 | 3-4 | <3 |
| 存货周转率 | >5 | 3-5 | 2-3 | <2 |

## 七、应急处理流程

### 突发情况应对

```
【现金即将断流】
1. 立即暂停所有投资
2. 申请短期贷款（优先级最高）
3. 若贷款额度已满，申请紧急贷款（接受扣分）
4. 减少下季度订单量

【报税时间不够】
1. CEO 提醒"CFO 注意时间"
2. 简化其他操作，优先保证报税
3. 若确实来不及，准备接受 5 分扣分

【电脑/网络故障】
1. 立即举手示意裁判
2. 申请备用机器
3. 队友协助记录当前状态
4. 恢复后快速确认数据

【队友操作失误】
1. 保持冷静，不指责
2. 记录失误影响
3. 后续季度弥补
4. 比赛结束后复盘
```
