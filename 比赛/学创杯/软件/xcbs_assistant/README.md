## 🚀 快速开始

### 1. 安装依赖

```bash
cd 比赛/学创杯/软件/xcbs_assistant
pip3 install -r requirements.txt
```

### 2. 启动应用

```bash
streamlit run app.py --server.port 8502
```

### 3. 访问系统

浏览器打开：`http://localhost:8502`

## 📋 比赛评分公式

```
总成绩 = 盈利能力 (30 分) + 营运能力 (25 分) + 偿债能力 (20 分) + 发展能力 (15 分) + 预算能力 (10 分) - 违规扣分
```

### 三条死亡红线

- 🚨 紧急借款：**-5 分/次**
- 🚨 逾期报税：**-5 分/次**
- 🚨 现金断流：**-10 分/次**

### 获奖标准

| 等级 | 分数要求 | 关键指标 |
|------|---------|---------|
| 🥇 一等奖 | ≥85 分 | 净利润率≥10%, ROE≥20%, 流动比率 1.2-1.5 |
| 🥈 二等奖 | 70-84 分 | 净利润率 7-10%, 流动比率≥1.0 |
| 🥉 三等奖 | 60-69 分 | 净利润率 5-7%, 无重大扣分 |

## 🛠️ 使用场景

### 赛前准备

1. **团队组建** → 使用【团队协作】设置角色分工
2. **熟悉规则** → 阅读【帮助中心】→ 比赛规则
3. **策略规划** → 使用【智能决策建议】→ 赛前策略规划
4. **下载模板** → 【模板中心】下载比赛用表
5. **物资准备** → 打印功能清单和使用指南

### 赛中决策（每季度）

| 时机 | 使用模块 | 负责人 |
|------|---------|--------|
| 季度初 | 现金流预测、预算编制助手 | CFO |
| 投融资决策 | 贷款决策助手 | CEO+CFO |
| 竞标前 | 报价策略计算器、广告 ROI 分析器 | CMO |
| 投资决策 | 产能规划工具 | COO |
| 季度末 | 财务比率计算器 | CFO |
| 全程 | 快捷工具箱（计时器） | 全员 |

### 赛后复盘

1. **数据记录** → 【比赛复盘记录表】
2. **趋势分析** → 【智能决策建议】→ 赛后复盘分析
3. **经验总结** → 【比赛复盘记录表】→ 复盘总结
4. **数据备份** → 【数据导入导出】→ 导出 Excel

## 💡 使用技巧

### 季度策略建议

| 季度 | 报价策略 | 广告投入 | 贷款策略 | 目标 |
|------|---------|---------|---------|------|
| Q1 | 0.85-0.90x | 8-10% | 积极 (≤65%) | 抢占市场 |
| Q2 | 0.95-1.00x | 5-7% | 稳健 (≤60%) | 稳定经营 |
| Q3 | 0.95-1.05x | 精准 (ROI≥6) | 适度 (≤65%) | 扩张发展 |
| Q4 | 1.05-1.10x | 3-5% | 偿还债务 | 冲刺利润 |

### 关键指标预警线

| 指标 | 安全线 | 危险线 |
|------|-------|-------|
| 现金余额 | ≥固定支出×1.2 | <固定支出×0.8 |
| 流动比率 | ≥1.2 | <1.0 |
| 速动比率 | ≥0.8 | <0.6 |
| 资产负债率 | <60% | >70% |
| 产能利用率 | ≥85% | <70% |
