# 学创杯比赛辅助软件需求分析

## 📊 比赛核心需求

### 评分公式 (总分 100 分)
```
总成绩 = 盈利能力 (30 分) + 营运能力 (25 分) + 偿债能力 (20 分) + 发展能力 (15 分) + 预算能力 (10 分) - 违规扣分
```

### 三条死亡红线
1. **紧急借款** - 扣 5 分/次
2. **逾期报税** - 扣 5 分/次  
3. **现金断流** - 扣 10 分/次

---

## 🎯 可开发的辅助软件

### 1. 💰 现金流预测工具 (最高优先级 ⭐⭐⭐⭐⭐)
**功能**:
- 输入期初现金、预计销售回款、采购支出、费用支出
- 自动计算期末现金余额
- 现金预警（低于安全线时报警）
- 贷款需求测算

**获奖价值**: 避免现金断流 (-10 分) 和紧急借款 (-5 分)

### 2. 📐 预算编制助手 (高优先级 ⭐⭐⭐⭐⭐)
**功能**:
- 销售收入预算（基于销量预测 + 价格策略）
- 生产成本预算（材料 + 人工 + 制造费用）
- 费用预算（管理费 + 销售费 + 财务费）
- 现金预算（收支汇总）
- 预算偏差率实时计算

**获奖价值**: 预算能力占 10 分，偏差<10% 才得高分

### 3. 🏷️ 报价策略计算器 (高优先级 ⭐⭐⭐⭐)
**功能**:
- 输入单位成本（材料 + 人工 + 制造费用）
- 选择策略：抢市场 (0.85x) / 正常 (1.0x) / 利润导向 (1.1x)
- 显示预计利润率
- 对比竞争对手报价
- 推荐最优报价区间

**获奖价值**: 直接影响销售收入和利润

### 4. 📊 财务比率实时计算器 (中高优先级 ⭐⭐⭐⭐)
**功能**:
- 输入资产负债表和利润表数据
- 自动计算 5 大能力 15 个指标
- 显示当前得分估算
- 提供改进建议（如：提升流动比率到 1.5）
- 与获奖标准对比（一等奖≥85 分）

**获奖价值**: 90 分来自财务指标计算

### 5. 📈 广告投放 ROI 分析器 (中优先级 ⭐⭐⭐)
**功能**:
- 输入广告投入金额
- 输入带来的订单金额
- 计算 ROI（目标≥1:5）
- 建议最优广告预算
- 历史 ROI 记录对比

**获奖价值**: 优化销售费用，提升利润率

### 6. 💳 贷款决策助手 (中优先级 ⭐⭐⭐)
**功能**:
- 短贷 vs 长贷对比
- 利息计算（短贷 6%/年，长贷 8%/年）
- 资产负债率预警（>70% 危险）
- 还款计划表
- 融资成本最小化建议

**获奖价值**: 优化资本结构，降低财务费用

### 7. 🏭 产能规划工具 (中优先级 ⭐⭐⭐)
**功能**:
- 输入设备数量和单台产能
- 计算总产能
- 产能利用率分析（目标≥90%）
- 投资回收期计算
- 新设备投资建议

**获奖价值**: 避免产能闲置，提升营运效率

### 8. 📝 比赛复盘记录表 (低优先级 ⭐⭐)
**功能**:
- 记录每轮决策数据
- 自动生成经营报告
- 与历史成绩对比
- 失误分析和改进建议

**获奖价值**: 快速提升训练效果

## 🚀 开发计划

### Phase 1: 核心功能 (比赛前必做)
1. **现金流预测工具** - 避免死亡红线
2. **财务比率计算器** - 实时估算得分
3. **预算编制助手** - 保证预算准确性

### Phase 2: 增强功能 (冲刺一等奖)
4. **报价策略计算器** - 优化收入
5. **贷款决策助手** - 优化资本结构
6. **产能规划工具** - 提升营运效率

### Phase 3: 辅助功能 (锦上添花)
7. **广告 ROI 分析器**
8. **比赛复盘记录表**

## 📋 软件技术要求

| 要求 | 方案 |
|------|------|
| **快速启动** | Streamlit (一键启动) |
| **离线使用** | 本地 SQLite 数据库 |
| **计算准确** | 内置比赛公式验证 |
| **操作简单** | 界面简洁，3 步完成计算 |
| **数据保存** | 自动保存每轮决策数据 |
| **可视化** | 仪表板显示关键指标 |
