## 第 2 周：单项技能突破（第 8-14 天）

### 🎯 本周目标

- 每个决策模块都能熟练操作
- 掌握核心计算公式
- 预算偏差控制在 20% 以内

### 📋 每日训练内容

#### 第 8 天：现金预算专项训练

**训练任务（2 小时）**
- [ ] 练习 5 次现金预算编制
- [ ] 目标：预算和实际偏差<20%
- [ ] 用 Excel 做辅助计算表

**计算公式**
```
可用资金 = 期初现金 + 上期回款
固定支出 = 工资 + 租金 + 费用
安全现金 = 固定支出 × 1.2
```

**评分标准**
| 偏差率 | 等级 |
|--------|------|
| <10% | 优秀 |
| 10-20% | 良好 |
| 20-30% | 合格 |
| >30% | 需改进 |

---

#### 第 9 天：贷款决策专项训练

**训练任务（2 小时）**
- [ ] 练习 5 次贷款决策
- [ ] 目标：资金够用，利息最低
- [ ] 计算不同贷款组合的利息成本

**决策要点**
```
什么时候需要贷款？
→ 现金缺口 = 支出 - 可用资金 > 0

选择短期还是长期？
→ 临时周转用短期，扩大产能用长期

贷款多少合适？
→ 资金缺口 + 安全边际，不超额贷款
```

---

#### 第 10 天：设备投资专项训练

**训练任务（2 小时）**
- [ ] 练习 5 次设备投资决策
- [ ] 目标：产能利用率>85%
- [ ] 计算设备投资回报率

**决策要点**
```
何时买设备？
→ 订单量 > 现有产能 × 1.2

买多少？
→ 新增产能 = (预计订单 - 现有产能) ÷ 单条线产能

投资回报计算：
设备投资回报 = 新增订单利润 ÷ 设备投资额
理想值：>30%
```

---

#### 第 11 天：报价策略专项训练

**训练任务（2 小时）**
- [ ] 练习 10 次报价决策
- [ ] 目标：中标率 60-80%
- [ ] 记录不同报价的中标情况

**报价策略**
```
成本计算：
单位成本 = 材料 + 人工 + 制造费用

报价公式：
抢占市场：成本 × 0.85-0.90
正常报价：成本 × 0.95-1.05
利润导向：成本 × 1.05-1.10
```

---

#### 第 12 天：广告投放专项训练

**训练任务（2 小时）**
- [ ] 练习 5 次广告投放
- [ ] 目标：广告 ROI > 1:5
- [ ] 记录不同投入带来的订单量

**决策要点**
```
广告投入多少？
→ 目标订单金额 ÷ 6 = 合理广告费

什么时候多投？
→ 第 1 季度抢市场、产能充足时

什么时候少投？
→ 产能不足、现金紧张时
```

---

#### 第 13 天：报税专项训练

**训练任务（1 小时）**
- [ ] 练习 5 次报税流程
- [ ] 目标：5 分钟内完成，0 错误
- [ ] 熟练计算增值税和所得税

**报税公式**
```
增值税 = 销项税额 - 进项税额
所得税 = 利润总额 × 25%

注意：
- 每季度必须报税
- 最后一个季度可以合理避税（控制利润）
```

## 第 3 周：双人配合训练（第 15-21 天）

### 🎯 本周目标

- CFO + 运营两人默契配合
- 决策时间缩短 30%
- 完成 3 次双人模拟

### 📋 训练安排

#### 第 15、17、19 天：双人模拟（隔天训练）

**训练模式（每次 3 小时）**
```
14:00-14:30  理论讲解 + 分工确认
14:30-16:30  完整模拟（4 季度）
16:30-17:00  复盘讨论
```

**角色分工**

**CFO 负责：**
- 全面预算编制
- 现金预算
- 贷款申请和还款
- 报税
- 财务报表分析

**运营负责：**
- 厂房/设备决策
- 采购决策
- 生产投料
- 人员招聘
- 订单交付

**CEO 负责：**
- 观察两人配合
- 记录决策时间
- 协调分歧
- 最终拍板（有争议时）

**配合流程**
```
每季度决策顺序：

1. CFO：编制预算（5 分钟）
   ↓
2. CFO+运营：讨论资金分配（5 分钟）
   ↓
3. 运营：提出产能需求（3 分钟）
   ↓
4. CFO：确认可用资金（2 分钟）
   ↓
5. 运营：执行采购/生产（10 分钟）
   ↓
6. 运营：报价和广告（5 分钟）
   ↓
7. CFO：报税（5 分钟）
   ↓
8. 全员：复盘（5 分钟）
```

**策略讨论要点**
```

1. 对手都很激进（拼命借钱扩张）
   → 我们保持稳健，等对手出错

2. 对手都很保守（不怎么投资）
   → 我们适度扩张，抢占市场

3. 价格战（对手疯狂低价抢单）
   → 我们避开锋芒，做差异化
```

## 💡 训练小贴士

### 高效训练技巧

1. **固定训练时间**：每天同一时间训练，形成生物钟
2. **训练前热身**：5 分钟复习公式和操作
3. **训练后复盘**：至少花 15 分钟总结
4. **交叉训练**：不要只练一个模块，轮换着来
5. **休息很重要**：训练 1 小时休息 10 分钟

### 避免训练倦怠

1. **设定小目标**：每天一个小目标，完成后打勾
2. **记录进步**：每周对比成绩，看到自己的成长
3. **团队激励**：互相鼓励，不抱怨不甩锅
4. **适当放松**：训练之余聊天放松，不给自己太大压力
