## 二、评分公式详解

### 2.1 总成绩计算

```
总成绩 = 软件最终成绩 - 违规扣分

软件最终成绩 = (第 1 季度综合分 + 第 2 季度综合分 + 第 3 季度综合分 + 第 4 季度综合分) ÷ 4

每季度综合表现分数 = 盈利能力 + 营运能力 + 偿债能力 + 发展能力 + 预算能力 - 报税扣分 - 紧急借款扣分

违规扣分 = 违规次数 × 5 分/次
```

### 2.2 五大核心能力指标

| 能力维度 | 权重 | 关键指标 |
|---------|------|---------|
| **盈利能力** | 30% | 净利润率、净资产收益率、总资产报酬率 |
| **营运能力** | 25% | 应收账款周转率、存货周转率、总资产周转率 |

## 五、核心决策技巧

### 5.1 资金管理"三不"原则

1. **不发生紧急借款**（每次扣 5 分，影响偿债能力）
2. **不逾期报税**（每次扣 5 分，影响信用评分）
3. **不资金链断裂**（现金余额不能为负）

### 5.2 现金流预测模型

```
期末现金 = 期初现金 + 销售回款 + 融资流入 - 采购支出 - 费用支出 - 还本付息

安全现金余额 ≥ 下季度固定支出 × 1.2
```

### 5.3 投资决策优先级

```
第 1 优先级：生产线设备（直接影响产能和交付）
第 2 优先级：厂房购置（避免租金浪费，考虑折旧抵税）
第 3 优先级：研发投入（提升产品竞争力）
第 4 优先级：市场开拓（长期收益，短期谨慎）
```

### 5.4 订单报价策略

**动态平衡法公式：**

```
最优报价 = 单位成本 × (1 + 目标利润率) × 竞争系数

竞争系数 = 1.0（市场平均）
         = 0.9（抢占市场）
         = 1.1（利润导向）
```

**报价技巧：**
- Q1：低价策略（0.85-0.9 倍），快速抢占市场份额
- Q2-Q3：动态调整（0.95-1.05 倍），平衡利润与销量
- Q4：利润导向（1.05-1.1 倍），冲刺净利润指标

### 5.5 广告投放 ROI 计算

```
广告投入产出比 = 新增订单金额 ÷ 广告投放额

理想区间：1:5 ~ 1:8
（即每投入 1 元广告费，带来 5-8 元订单）
```

## 六、避坑指南（扣分项）

### 6.1 常见违规扣分

| 违规行为 | 扣分 | 避免方法 |
|---------|------|---------|
| 紧急借款 | 5 分/次 | 做好现金预算，留足安全边际 |
| 逾期报税 | 5 分/次 | 每季度最后 5 分钟必须完成报税 |
| 订单违约 | 5 分/单 | 生产投料前确认产能，不接超额订单 |
| 现金断流 | 10 分/次 | 设置现金预警线（≥下期固定支出 120%） |
| 违规操作 | 5 分/次 | 严格遵守赛场纪律 |

### 6.2 财务指标陷阱

| 陷阱 | 后果 | 应对 |
|------|------|------|
| 资产负债率 > 70% | 偿债能力得分骤降 | 控制贷款规模，适时股权融资 |
| 流动比率 < 1.0 | 短期偿债风险高 | 保持现金 + 应收账款 > 短期负债 |
| 存货积压 | 营运能力得分降低 | 按需生产，Q4 主动去库存 |
| 应收账款账期过长 | 现金流紧张 | 优先接回款周期短的订单 |

## 附录：核心财务公式速查

### A. 盈利能力指标
```
净利润率 = 净利润 ÷ 销售收入 × 100%
净资产收益率(ROE) = 净利润 ÷ 平均净资产 × 100%
总资产报酬率(ROA) = 利润总额 ÷ 平均资产总额 × 100%
```

### B. 营运能力指标
```
应收账款周转率 = 销售收入 ÷ 平均应收账款
存货周转率 = 销售成本 ÷ 平均存货
总资产周转率 = 销售收入 ÷ 平均资产总额
```

### C. 偿债能力指标
```
流动比率 = 流动资产 ÷ 流动负债
速动比率 = (流动资产 - 存货) ÷ 流动负债
资产负债率 = 负债总额 ÷ 资产总额 × 100%
```

### D. 发展能力指标
```
销售增长率 = 本期销售增长额 ÷ 上期销售收入 × 100%
资本积累率 = 本期所有者权益增长额 ÷ 期初所有者权益 × 100%
总资产增长率 = 本期总资产增长额 ÷ 期初资产总额 × 100%
```
