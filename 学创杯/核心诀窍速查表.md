## 二、关键财务指标（必背公式）

### 📊 五大能力指标计算

#### 1. 盈利能力（30% 权重）

```
① 净利润率 = 净利润 ÷ 销售收入 × 100%
   优秀：>10%  良好：7-10%  合格：5-7%

② 净资产收益率 (ROE) = 净利润 ÷ 平均净资产 × 100%
   优秀：>20%  良好：15-20%  合格：10-15%

③ 总资产报酬率 (ROA) = 利润总额 ÷ 平均资产总额 × 100%
   优秀：>15%  良好：10-15%  合格：8-10%
```

#### 2. 营运能力（25% 权重）

```
① 应收账款周转率 = 销售收入 ÷ 平均应收账款
   优秀：>6 次  良好：4-6 次  合格：3-4 次

② 存货周转率 = 销售成本 ÷ 平均存货
   优秀：>5 次  良好：3-5 次  合格：2-3 次

③ 总资产周转率 = 销售收入 ÷ 平均资产总额
   优秀：>1.5  良好：1.0-1.5  合格：0.8-1.0
```

#### 3. 偿债能力（20% 权重）

```
① 流动比率 = 流动资产 ÷ 流动负债
   优秀：>1.5  良好：1.2-1.5  合格：1.0-1.2  危险：<1.0

② 速动比率 = (流动资产 - 存货) ÷ 流动负债
   优秀：>1.2  良好：0.8-1.2  合格：0.6-0.8

③ 资产负债率 = 负债总额 ÷ 资产总额 × 100%
   优秀：<50%  良好：50-60%  合格：60-70%  危险：>70%
```

#### 4. 发展能力（15% 权重）

```
① 销售增长率 = 本期销售增长额 ÷ 上期销售收入 × 100%
   优秀：>20%  良好：10-20%  合格：5-10%

② 资本积累率 = 本期所有者权益增长额 ÷ 期初所有者权益 × 100%
   优秀：>15%  良好：10-15%  合格：5-10%

③ 总资产增长率 = 本期总资产增长额 ÷ 期初资产总额 × 100%
   优秀：>15%  良好：10-15%  合格：5-10%
```

#### 5. 预算能力（10% 权重）

```
预算偏差率 = |实际值 - 预算值| ÷ 预算值 × 100%

优秀：<5%  良好：5-10%  合格：10-15%  需改进：>15%
```

## 五、常见场景应对策略

### 🚨 场景 1：现金紧张

**预警信号**：
- 现金余额 < 下期固定支出×1.2
- 流动比率 < 1.2

**应对措施**：
```
立即行动（按优先级）：
1. 暂停所有投资（厂房/设备/研发）
2. 减少广告投放（砍掉 50%）
3. 申请短期贷款（优先于长贷）
4. 折价促销（加速回款）
5. 延期付款（与供应商协商）

极端情况：
→ 申请紧急贷款（接受 5 分扣分，避免断流扣 10 分）
```

---

### 🚨 场景 2：产能过剩

**预警信号**：
- 产能利用率 < 70%
- 单位产品折旧成本高

**应对措施**：
```
1. 低价接单（填补产能）
   报价 = 变动成本 + 少量利润

2. 停止新设备投资
   已有设备最大化利用

3. 减少生产人员
   自然流失不补充

4. 考虑转产
   生产其他高需求产品
```

---

### 🚨 场景 3：产能不足

**预警信号**：
- 订单接不下（中标率>90% 但交货不足）
- 产能利用率 > 95%

**应对措施**：
```
1. 优先高利润订单
   筛选利润率>15% 的订单

2. 投资新设备
   投资回报计算：
   新增利润 = (新订单量 - 现有产能) × 单位利润
   投资回收期 = 设备投资 ÷ 新增利润
   回收期<2 年可投资

3. 优化生产流程
   减少换线时间
   提升生产效率

4. 适度外包
   部分工序外包（成本高但保交付）
```

---

### 🚨 场景 4：价格战

**预警信号**：
- 竞争对手报价低于成本价
- 中标率骤降

**应对措施**：
```
策略选择：

方案 A：跟随降价（适合资金充足）
- 报价降至对手水平
- 但控制接单量（避免亏本太多）
- 等待对手资金链断裂

方案 B：差异化竞争（推荐）
- 不参与价格战
- 专注高利润细分市场
- 提升产品质量/服务

方案 C：战略撤退
- 放弃部分市场
- 保存實力
- 等价格战结束再反攻
```

---

### 🚨 场景 5：时间不够用

**预警信号**：
- 剩余时间 < 15 分钟，还未报税
- 决策流程混乱

**应对措施**：
```
立即启动应急预案：

1. CEO 喊"时间预警"
2. CFO 暂停其他工作，准备报税
3. 简化决策流程（抓大放小）
4. 用速查表快速计算
5. 先完成再优化

报税优先级最高（5 分不能丢）！
```

## 六、快速计算技巧（手算版）

### 🔢 心算/计算器技巧

#### 1. 报价快速计算

```
成本价 = 材料 + 人工 + 制造费用

例：材料 30 + 人工 20 + 制造费用 10 = 60 元

抢占市场价：60 × 0.9 = 54 元
正常价：60 × 1.0 = 60 元
利润价：60 × 1.1 = 66 元
```

#### 2. 贷款利息估算

```
短期贷款利息 = 贷款金额 × 年利率 ÷ 4（季度）
例：100 万 × 6% ÷ 4 = 1.5 万/季度

长期贷款利息 = 贷款金额 × 年利率 ÷ 4
例：200 万 × 8% ÷ 4 = 4 万/季度
```

#### 3. 现金流预测

```
期末现金 = 期初现金 + 销售回款 + 贷款 - 支出

快速估算法：
下月可用资金 = 本月现金×0.8 + 下月回款×0.7

（假设 20% 用于固定支出，30% 回款延迟）
```

#### 4. 广告 ROI 估算

```
目标订单金额 = 产能 × 单价
合理广告费 = 目标订单金额 ÷ 6

例：产能 1000 件 × 单价 100 元 = 100 万订单
广告费 = 100 万 ÷ 6 ≈ 17 万
```

#### 5. 利润率倒推报价

```
目标利润率 10%，成本 60 元

报价 = 成本 ÷ (1 - 利润率)
     = 60 ÷ (1 - 0.1)
     = 60 ÷ 0.9
     = 66.67 元
```

## 七、团队协作技巧

### 🤝 分工明确

| 角色 | 决策权 | 执行权 | 建议权 |
|------|--------|--------|--------|
| **CEO** | 战略方向、重大投资 | 协调配合 | 所有决策 |
| **CFO** | 预算、贷款、报税 | 财务操作 | 经营决策 |
| **运营** | 产能、采购、生产 | 经营操作 | 财务决策 |

### 💬 沟通暗号

| 暗号 | 含义 | 响应 |
|------|------|------|
| "现金预警" | 现金可能不够 | 暂停投资，重新测算 |
| "时间到了" | 该报税了 | CFO 立即报税 |
| "再算一遍" | 数据可疑 | 全员停止，复核数据 |
| "可以" | 同意 | 继续执行 |
| "再想想" | 有风险 | 讨论替代方案 |
| "我负责" | 主动承担责任 | 其他人配合 |

### ⚡ 争议解决机制

```
当团队意见不一致时：

1. CEO 听取双方意见（1 分钟）
2. 快速风险评估（30 秒）
3. CEO 拍板决定（不可更改）
4. 全员执行（不抱怨）
5. 赛后复盘（不甩锅）
```

## 十、最后叮嘱

### ⭐ 获奖三要素

```
1. 不扣分（紧急借款 0 + 逾期报税 0 + 现金断流 0）
2. 预算准（偏差<10%）
3. 团队默契（不争吵，高效决策）
```

### ❌ 常见失败原因

```
1. 现金管理不当 → 紧急借款/断流
2. 忘记报税 → 白送 5 分
3. 盲目扩张 → 产能闲置，利润下降
4. 报价失误 → 要么没订单，要么亏本
5. 团队内讧 → 决策效率低，错失良机
6. 时间管理差 → 最后一团糟
```

### ✅ 成功秘诀

```
稳健第一，策略第二
```
