## 🚀 快速开始

### 1. 安装依赖

```bash
cd 软件开发/财务工具箱
pip3 install -r requirements.txt
```

### 2. 启动应用

```bash
streamlit run app.py --server.port 8501
```

### 3. 访问系统

浏览器打开：`http://localhost:8501`

**默认账户**:
- 用户名：`admin`
- 密码：`admin123`
