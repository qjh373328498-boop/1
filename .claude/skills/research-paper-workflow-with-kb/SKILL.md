# 学术研究/论文写作工作流（知识库参考模式）

**⚠️ 触发条件**：

仅当用户**明确表述**以下意思时使用此工作流：
- ✅ "可以参考知识库"
- ✅ "用知识库的资料帮我..."
- ✅ "结合我之前的资料..."
- ✅ "参考我之前的学习笔记"
- ✅ "用我比赛的那些资料"

**否则，默认使用隔离模式**（`research-paper-workflow-isolated`）

## 完整工作流程

### 阶段 0：知识库扫描（仅此一模式）

**0.1 确认用户授权**
```
向用户确认：
"好的，我将参考您的知识库资料。请问：
1. 需要参考哪些特定目录？（如：会计专业/比赛资料/课程笔记）
2. 还是需要扫描全部知识库？"
```

**0.2 扫描相关知识库目录**
```
读取路径：
- 知识库/01-个人文档/个人学业档案索引.md
- 知识库/02-学习资料/
- 知识库/03-项目档案/（比赛资料等）
- 知识库/04-模板库/

提取：
- 相关主题的个人笔记
- 比赛项目资料
- 课程作业中的相关内容
- 之前研究的积累
```

**0.3 知识库内容可信度评估**
```
标记来源可信度：
- 🔴 个人笔记/理解（需标注"个人整理"）
- 🟡 课程作业（需标注"课程作业，未经 peer review"）
- 🟢 比赛报告（相对可靠，基于实际项目）
- ✅ 外部权威文献（最高优先级引用）
```

### 阶段 1：资料收集与提取

**资料来源（按优先级）**：
1. ✅ 外部权威文献（学术数据库、官方报告）
2. ✅ 用户明确提供的文件
3. 🟡 知识库中的比赛资料/项目报告
4. 🟡 知识库中的课程作业（高分作业优先）
5. 🔴 知识库中的个人笔记（仅作参考，不可直接引用）

**1.1 网页资料**
```
调用：baoyu-url-to-markdown
- 读取 URL 内容
- 转换为干净 Markdown
- 保存 HTML 快照
```

**1.2 YouTube 视频**
```
调用：baoyu-youtube-transcript
- 下载字幕/讲义
- 支持多语言
- 可选章节划分
```

**1.3 PDF/Word/Excel/PPT**
```
调用：pdf/docx/xlsx/pptx skills
- 提取文档内容
- 转换为目标格式
```

**1.4 知识库文档**
```
读取 Markdown 文件：
- 个人学业档案索引
- 会计专业学习与职业发展指南
- 比赛资料索引
- 学科竞赛备赛资料索引
- 课程笔记
- 数据分析实战指南
```

### 阶段 2-5

与隔离模式相同（翻译→格式化→配图→导出）

## 快捷命令模式

```bash
# 完整流程（知识库模式）
/research-paper-workflow-with-kb <主题>

# 指定知识库目录
/research-paper-workflow-with-kb <主题> --kb-dir 知识库/02-学习资料/会计专业/

# 仅使用知识库
/research-paper-workflow-with-kb <主题> --kb-only

# 排除某些目录
/research-paper-workflow-with-kb <主题> --exclude 知识库/01-个人文档/
```

## 知识库引用规范

### 可引用的知识库内容

| 内容类型 | 引用方式 | 示例 |
|----------|----------|------|
| 比赛报告 | "根据作者参与的 XX 比赛项目报告（2025）..." | "学创杯财务决策模拟赛项报告" |
| 高分课程作业 | "参见 XX 课程优秀作业（成绩：XX）" | "中级财务会计课程论文" |
| 项目文档 | "根据 XX 项目文档" | "土地资源实践创新大赛项目文档" |
| 个人笔记 | "个人整理，基于 XX 来源" | "基于《会计研究》2025 年第 X 期的个人理解" |

### 不可直接引用的内容

- ❌ 未经核实的数据
- ❌ 个人主观理解（无外部来源支撑）
- ❌ 草稿、未完成的文档
- ❌ 学习笔记中的简化表述

### 引用标注格式

```markdown
## 参考文献

### 外部权威来源（优先级高）
[1] 作者。论文标题 [J].期刊名，年份，卷 (期): 页码.
[2]  governmental report...

### 知识库来源（需特殊标注）
[3] 作者。个人学业档案 [R/OL].广工知识库，2025. 
    （注：个人学习记录，非 peer-reviewed 来源）
[4] 作者。XX 比赛项目报告 [R]. 知识库，2025.
    （注：基于实际参赛经历的项目文档）
```

## 使用示例

### 示例 1：写课程论文（允许参考知识库）

用户："帮我写一篇环境会计的论文，可以参考知识库"

工作流：
1. ✅ 扫描知识库中的会计专业资料
2. ✅ 读取"会计专业学习与职业发展指南"
3. ✅ 参考"数据分析实战指南"中的会计分析方法
4. ✅ 搜索外部学术文献（知网等）
5. 整合知识库资料和外部文献
6. 明确标注哪些来自知识库（非 peer-reviewed）
7. 导出 Word

### 示例 2：比赛总结报告（使用个人资料）

用户："用我之前的比赛资料写个总结报告"

工作流：
1. ✅ 读取知识库/03-项目档案/比赛资料索引.md
2. ✅ 提取学创杯、土地资源大赛的项目文档
3. ✅ 整合个人比赛记录和成绩
4. ✅ 生成总结报告
5. 标注"基于作者实际参赛经历"

### 示例 3：学习笔记整理

用户："结合我的知识库整理一份会计分录的复习资料"

工作流：
1. ✅ 读取知识库中的中级财会笔记
2. ✅ 提取个人学业档案中的相关记录
3. ✅ 参考会计专业学习指南
4. 整合为复习文档
5. 标注"个人学习整理，仅供参考"

## 注意事项

### 知识库使用原则

1. **明确授权**：必须用户明确说才能使用
2. **可信度标记**：区分外部文献和个人资料
3. **引用规范**：知识库内容标注为非 peer-reviewed
4. **优先级**：外部权威文献 > 比赛报告 > 课程作业 > 个人笔记
5. **不捏造**：不夸大知识库内容的权威性

### 适用场景

| 场景 | 推荐模式 | 理由 |
|------|---------|------|
| 正式学术论文 | 隔离模式 | 确保学术诚信 |
| 课程论文 | 知识库模式 | 可展示个人学习积累 |
| 比赛报告 | 知识库模式 | 基于实际参赛经历 |
| 学习笔记 | 知识库模式 | 整理个人知识体系 |
| 草稿/初稿 | 知识库模式 | 快速整合现有资料 |

## 与隔离模式的区别

| 特性 | 隔离模式（默认） | 知识库模式（需明确） |
|------|-----------------|---------------------|
| 触发条件 | 默认 | 用户明确说"参考知识库" |
| 资料来源 | 仅外部文献 + 用户提供 | 外部文献 + 知识库 + 用户提供 |
| 知识库读取 | ❌ 禁止 | ✅ 允许 |
| 引用个人笔记 | ❌ 不允许 | ✅ 允许（需标注） |
| 适用场景 | 正式论文、竞赛 | 课程作业、学习整理 |
| 学术诚信级别 | 高 | 中（需标注来源性质） |
