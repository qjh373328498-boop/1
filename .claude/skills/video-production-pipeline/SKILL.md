# 🎬 一体化视频生成工作流

## 描述

整合从脚本→配音→画面→字幕→BGM→最终合成的完整视频制作流程。

**核心理念**：一句话描述需求，全自动生成完整视频。

## 触发关键词

- `做视频` / `生成视频` / `制作视频`
- `从头做个视频` / `一体化视频`
- `产品介绍视频` / `宣传片` / `vlog`

## 完整流程

```
┌─────────────────────────────────────────────────────────────┐
│                    用户输入一句话                            │
│          "帮我做一个产品介绍视频，60 秒，科技感"                │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 1: 脚本生成（AI 文案）                                │
│  - 分析视频类型（产品介绍）                                 │
│  - 确定时长（60 秒）                                        │
│  - 生成脚本结构                                             │
│  输出：script.txt                                           │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 2: 配音生成（tts-voice）                              │
│  - 选择语音（科技感 → Yunxi 阳光男声）                       │
│  - 生成配音和字幕                                           │
│  输出：narration.mp3 + narration.vtt                        │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 3: 画面生成（hyperframes）                            │
│  - 根据脚本生成 HTML 描述                                    │
│  - 渲染成视频画面                                           │
│  输出：visuals.mp4                                          │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 4: BGM 生成（music-gen）                               │
│  - 选择风格（科技感 → Technology/Electronic）                │
│  - 生成背景音乐                                             │
│  输出：bgm.mp3                                              │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 5: 最终合成（video-use + FFmpeg）                      │
│  - 合并画面 + 配音+BGM                                       │
│  - 添加字幕                                                 │
│  - 音量平衡调整                                             │
│  输出：final_video.mp4                                      │
└─────────────────────────────────────────────────────────────┘
```

## 预设模板

### 模板 1：产品介绍（竖屏 60 秒）
```yaml
适用场景：抖音/小红书/视频号
脚本结构：
  - 0-5s:  痛点引入（"你是不是也遇到过..."）
  - 5-20s: 产品介绍（"这就是我们的..."）
  - 20-45s: 功能展示（3 个核心卖点）
  - 45-60s: 行动号召（"立即体验"）
配音：zh-CN-XiaoxiaoNeural（温暖女声）
画面：产品图片+文字动画+数据图表
BGM：Corporate Uplifting，音量 25%
字幕：大字，底部，关键字高亮
```

### 模板 2：知识科普（横屏 180 秒）
```yaml
适用场景：YouTube/B 站/抖音
脚本结构：
  - 0-10s:  问题提出
  - 10-30s: 背景介绍
  - 30-120s: 核心知识点（3-5 个）
  - 120-150s: 案例说明
  - 150-180s: 总结回顾
配音：zh-CN-YunyangNeural（专业男声）
画面：图文动画+数据可视化
BGM：Ambient Documentary，音量 15%
字幕：双语，可切换
```

### 模板 3：Vlog 日常（竖屏 30 秒）
```yaml
适用场景：朋友圈/小红书/抖音
脚本结构：
  - 0-3s:  吸引注意（"今天带你去..."）
  - 3-25s: 核心内容
  - 25-30s: 互动引导（"你觉得呢？"）
配音：zh-CN-YunxiaNeural（可爱男声）
画面：真人出镜+转场特效
BGM：轻快 Pop，音量 20%
字幕：关键字高亮，表情符号
```

## 快速使用

### 方式 1：一句话生成
```
用户："帮我做一个产品介绍视频，60 秒，科技感"

自动执行：
1. 生成脚本
2. 调用 tts-voice 生成配音
3. 调用 hyperframes 生成画面
4. 调用 music-gen 生成 BGM
5. 调用 video-use 合成最终视频
```

### 方式 2：自定义参数
```
用户："生成视频，主题 [人工智能科普], 时长 [3 分钟], 风格 [专业], 配音 [男声]"

自动匹配：
- 脚本结构：知识科普模板
- 配音：zh-CN-YunyangNeural
- BGM: Documentary
- 画面：图文动画
```

### 方式 3：分步确认
```
用户："我想做个视频"

AI: "好的，请告诉我：
     1. 视频主题是什么？
     2. 预计时长多久？
     3. 目标平台是？（抖音/YouTube/B 站）"

根据回答生成脚本 → 用户确认 → 开始制作
```

## 质量检查清单

### 视频质量
- [ ] 分辨率：1080x1920（竖屏）或 1920x1080（横屏）
- [ ] 帧率：30fps
- [ ] 码率：>8Mbps

### 音频质量
- [ ] 配音清晰（-16LUFS）
- [ ] BGM 不压过人声（-20dB 以下）
- [ ] 无爆音/杂音

### 字幕质量
- [ ] 无错别字
- [ ] 时间轴准确（误差<100ms）
- [ ] 可读性强（字号、颜色、对比度）

### 内容质量
- [ ] 开场 3 秒有吸引力
- [ ] 信息密度合理
- [ ] 结尾有行动号召

- 存储：SSD（读写速度影响大）

## 能力限制

### 当前限制
- ❌ 无法访问外部 URL 下载素材
- ❌ 无真实图片/视频素材库
- ❌ 渲染速度较慢（3 分钟视频约需 30 分钟）
- ❌ 文字动画效果有限
- ⚠️  中文字体需手动安装

### 推荐方案
对于**正式出品的视频**，强烈推荐混合工作流：

| 方案 | 流程 | 优点 | 缺点 | 适用场景 |
|------|------|------|------|----------|
| **A. 快速原型** | 全自动本地生成 | 快速、无人工 | 画面简单 | 内部审核 |
| **B. 混合制作** ⭐ | 配音 + 剪映 | 专业、高效 | 需 10 分钟人工 | 正式出品 |
| **C. 专业制作** | 真人 + 专业剪辑 | 电影级 | 成本高 | 商业项目 |
