# 比赛备赛工作流

专为大学生学科竞赛设计的完整备赛流程，整合了规则解读、备赛计划、材料准备、PPT 优化、答辩辅导等功能。

## 触发场景

当用户提到以下需求时自动触发：
- "参加 XX 比赛"、"准备 XX 赛"、"XX 杯备赛"
- "写比赛计划书"、"做路演 PPT"、"准备答辩"
- "比赛规则是什么"、"备赛计划"

## 支持的比赛类型

| 比赛 | 类型 | 特点 |
|------|------|------|
| 学创杯 | 财务决策模拟 | 软件操作 + 战略决策 |
| 挑战杯 | 创业计划 | 商业计划书 + 路演答辩 |
| 国创赛 | 创新创业 | 项目计划书 + 展示 |
| 互联网+ | 创新创业 | 商业计划 + 实践 |
| 案例分析大赛 | 案例分析 | 分析报告 + 展示 |
| 数学建模 | 建模竞赛 | 论文 + 模型 |

## 阶段 1：比赛规则解读

**1.1 读取比赛官方文件**
```
资料来源：
- 比赛官网规则
- 往届获奖作品
- 知识库中的备赛资料
- YouTube 比赛讲解视频 ★新增
```

**1.2 提取关键信息**
```markdown
# {比赛名} 关键信息

## 基本信息
- **报名截止**：{日期}
- **提交截止**：{日期}
- **比赛时间**：{日期}
- **比赛形式**：□ 线上 □ 线下 □ 混合
- **团队人数**：{X}-{Y} 人
- **指导老师**：{数量} 名

## 评审标准
| 评分项 | 权重 | 要点 |
|--------|------|------|
| 创新性 | 30% | 创意新颖、技术先进 |
| 可行性 | 25% | 方案可行、落地性强 |
| 商业价值 | 25% | 市场前景、盈利模式 |
| 团队能力 | 10% | 团队配置、执行能力 |
| 展示效果 | 10% | PPT 质量、答辩表现 |

## 提交材料
- [ ] 商业计划书（{X} 页以内）
- [ ] 路演 PPT（{X} 分钟）
- [ ] 辅助材料（视频/原型等）
- [ ] 报名表
```

## 阶段 4：PPT 优化

**4.1 路演 PPT 结构**
```markdown
# 路演 PPT 框架（10 分钟）

## 1. 开场（30 秒）
- 痛点引入
- 解决方案

## 2. 市场痛点（1 分钟）
- 问题描述
- 市场空间

## 3. 产品介绍（2 分钟）
- 产品功能
- 核心优势

## 4. 商业模式（2 分钟）
- 盈利模式
- 运营数据

## 5. 竞争分析（1.5 分钟）
- 竞争优势
- 壁垒

## 6. 团队介绍（1 分钟）
- 核心团队
- 顾问

## 7. 财务预测（1.5 分钟）
- 收入预测
- 融资需求

## 8. 展望（1 分钟）
- 发展规划
- 愿景
```

**4.2 PPT 优化建议**
```
优化要点：
- 每页不超过 30 字
- 用图表代替文字
- 关键数据加粗
- 统一视觉风格
- 动画简洁专业
```

## 阶段 5：答辩题库

**5.1 常见问题分类**
```markdown
# 答辩题库

## 市场类问题
- Q1: 你们的目标市场有多大？
- Q2: 如何获取用户？
- Q3: 竞争对手有哪些？

## 技术类问题
- Q4: 技术壁垒是什么？
- Q5: 如何保证技术领先？
- Q6: 是否申请专利？

## 商业模式类问题
- Q7: 如何盈利？
- Q8: 收入来源有哪些？
- Q9: 毛利率多少？

## 团队类问题
- Q10: 为什么你们能做成功？
- Q11: 团队股份如何分配？
- Q12: 核心成员是否全职？

## 财务类问题
- Q13: 需要融资多少？
- Q14: 资金如何使用？
- Q15: 预计何时盈利？
```

**5.2 答案模板**
```markdown
### Q: 你们的目标市场有多大？

**回答框架**：
1. 总体市场规模（TAM）
2. 可服务市场规模（SAM）
3. 可获得市场规模（SOM）

**示例回答**：
"根据 XX 报告，我们这个领域的总体市场规模是 XXX 亿（TAM）。
我们初期聚焦 XX 细分市场，可服务市场规模是 XX 亿（SAM）。
第一年我们目标是获取 X%的市场份额，即 XX 万（SOM）。"
```

**5.3 模拟答辩**
```markdown
## 模拟答辩流程

1. 抽题（随机 3-5 个问题）
2. 思考（30 秒）
3. 回答（2 分钟/题）
4. 反馈（评委点评）
5. 改进建议
```

## 使用示例

### 示例 1：学创杯备赛

用户："我要参加学创杯，怎么准备？"

工作流：
1. 读取学创杯规则文件
2. 生成备赛时间表（假设 3 个月）
3. 制定训练计划
4. 整理往届获奖策略
5. 生成答辩题库

### 示例 2：挑战杯备赛

用户："挑战杯商业计划书怎么写？"

工作流：
1. 读取挑战杯规则
2. 提供商业计划书模板
3. 参考知识库中的获奖作品
4. 生成 PPT 框架
5. 准备答辩问题

## 注意事项

1. **尽早开始**：至少提前 2-3 个月准备
2. **研究规则**：充分理解评分标准
3. **参考往届**：学习获奖作品
4. **反复打磨**：PPT 和演讲稿多修改
5. **模拟答辩**：提前练习答辩
