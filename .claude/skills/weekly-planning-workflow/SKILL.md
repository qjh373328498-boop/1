# 周计划与时间管理工作流

专为大学生设计的时间管理和计划执行工作流，整合了目标设定、周计划制定、每日执行、回顾总结等功能。

## 触发场景

当用户提到以下需求时自动触发：
- "制定周计划"、"这周的计划"
- "安排下周日程"、"时间管理"
- "本周目标"、"周回顾"
- "怎么做计划"、"计划模板"

## 阶段 1：目标设定

**1.1 目标分类体系**
```markdown
## 目标层级

### 长期目标（学期/年度）
- 学业目标：GPA 目标、考证计划
- 比赛目标：获奖目标
- 技能目标：学会 Python、英语过六级

### 中期目标（月度）
- 完成 XX 课程学习
- 准备 XX 比赛
- 读完 X 本书

### 短期目标（周度）
- 完成本周作业
- 复习 X 章内容
- 运动 X 次
```

**1.2 SMART 原则**
```markdown
## 目标设定原则

- **S**pecific（具体的）
  - ❌ "学好会计"
  - ✅ "中级财务会计成绩达到 85 分以上"

- **M**easurable（可衡量的）
  - ❌ "多运动"
  - ✅ "每周运动 3 次，每次 30 分钟"

- **A**chievable（可实现的）
  - ❌ "一周学完一本书"
  - ✅ "一周读完一本书的前 3 章"

- **R**elevant（相关的）
  - ❌ "学习编程"（与专业无关）
  - ✅ "学习 Python 数据分析"（与会计相关）

- **T**ime-bound（有时限的）
  - ❌ "找个实习"
  - ✅ "3 月 31 日前拿到实习 offer"
```

## 阶段 2：周计划制定

**2.1 周计划模板**
```markdown
# 第 X 周计划（{日期范围}）

## 🎯 本周重点（3 件最重要的事）
1. [ ] {重点 1} - 优先级：🔴
2. [ ] {重点 2} - 优先级：🔴
3. [ ] {重点 3} - 优先级：🔴

## 📚 学习目标
- [ ] 完成{课程名}第 X 章学习
- [ ] 完成{课程名}作业
- [ ] 复习{课程名}知识点

## 🏆 比赛/项目
- [ ] {任务 1}
- [ ] {任务 2}

## 💪 健康生活
- [ ] 运动 X 次
- [ ] 早睡早起（23:00 前）
- [ ] 喝水充足

## 📅 固定日程
| 时间 | 周一 | 周二 | 周三 | 周四 | 周五 | 周六 | 周日 |
|------|------|------|------|------|------|------|------|
| 8:00 |      |      |      |      |      |      |      |
| 10:00|      |      |      |      |      |      |      |
| 14:00|      |      |      |      |      |      |      |
| 16:00|      |      |      |      |      |      |      |
| 19:00|      |      |      |      |      |      |      |
```

**2.2 任务分解（WBS）**
```markdown
## 任务分解示例

### 大任务：完成中级财务会计作业
- [ ] 1. 复习课本第 5 章（30min）
- [ ] 2. 查看作业要求（10min）
- [ ] 3. 完成选择题（20min）
- [ ] 4. 完成计算题（60min）
- [ ] 5. 检查答案（20min）

### 大任务：准备学创杯比赛
- [ ] 1. 研究比赛规则（60min）
- [ ] 2. 团队分工讨论（30min）
- [ ] 3. 第一次模拟练习（120min）
- [ ] 4. 复盘总结（30min）
```

**2.3 时间估算与分配**
```markdown
## 时间分配建议

### 学习类任务
- 课堂学习：按课表
- 课后复习：每节课后 2-3 小时
- 作业完成：每科 2-4 小时
- 考试复习：提前 2 周，每天 4-6 小时

### 比赛/项目
- 每周固定 6-8 小时
- 赛前冲刺：每周 15-20 小时

### 休闲生活
- 运动：每周 3-5 次，每次 30-60min
- 社交：每周 2-3 次
- 娱乐：每天≤2 小时

### 时间分配比例
- 学习：60%
- 比赛/项目：20%
- 生活：15%
- 机动：5%
```

## 阶段 3：日程安排

**3.1 每日时间块**
```markdown
## 典型一天 schedule

### 早晨（7:00-8:00）
- 起床、洗漱
- 早餐
- 晨读/背单词

### 上午（8:00-12:00）
- 上课/学习
- 课间休息

### 中午（12:00-14:00）
- 午餐
- 午休（20-30min）

### 下午（14:00-18:00）
- 上课/学习
- 运动锻炼

### 晚上（19:00-22:00）
- 作业/复习
- 阅读/技能学习

### 睡前（22:00-23:00）
- 洗漱
- 明日计划
- 放松
```

## 注意事项

1. **计划要现实**：不要安排过多，留出缓冲时间
2. **优先级明确**：每天 3 件最重要的事
3. **灵活调整**：计划赶不上变化，允许调整
4. **坚持回顾**：每周至少回顾一次
5. **奖励自己**：完成目标后给自己奖励
