# 考试复习与备考工作流

专门为大学生考试复习设计的工作流，整合了知识点提取、思维导图生成、记忆卡片制作、习题整理等功能。

## 触发场景

当用户提到以下需求时自动触发：
- "帮我复习 XXX 课程"、"准备考试"、"要期末考试了"
- "整理知识点"、"做思维导图"、"制作记忆卡片"
- "整理错题"、"生成复习计划"、"划重点"
- "背XXX"、"记忆 XXX"

## 阶段 1：资料收集

**1.1 读取课程资料**
```
资料来源：
- 课程 PPT（调用 pptx skill）
- 课程笔记（Markdown 文件）
- 教材 PDF（调用 pdf skill）
- 作业题目（Word/PDF）
- YouTube 教程视频（调用 baoyu-youtube-transcript）★新增
```

**1.2 读取知识库相关资料**
```
读取路径：
- 知识库/02-学习资料/01-课程笔记/
- 知识库/02-学习资料/02-考试复习/
- 作业（论文或笔记）/{课程名}/
```

**1.3 YouTube 教程视频下载（新增）**
```bash
# 下载 YouTube 教学视频并提取字幕
baoyu-youtube-transcript \
  'https://youtube.com/watch?v=xxx' \
  --languages zh,en \
  --chapters \
  --output-dir 复习资料/{课程名}/视频教程/

# 输出：
# - 视频教程/transcript.md（带章节的字幕）
# - 视频教程/cover.jpg（视频封面）
# - 视频教程/meta.json（视频元数据）
```

**用途**：
- 下载名校公开课
- 提取知识点讲解视频
- 获取实操演示视频
- 多语言字幕学习

## 阶段 2：知识点提取

**2.1 自动提取重点**
```
调用：baoyu-format-markdown + 自定义处理

提取规则：
- 标题层级分析（第 1-2 级标题为章节框架）
- 关键词提取（频率高、加粗、斜体）
- 定义/公式/定理识别
- 例题和案例分析
```

**2.2 知识点分类**
```markdown
## 知识点分类

### 理解类（需要深入理解）
- [知识点 1]：[解释]
- [知识点 2]：[解释]

### 记忆类（需要背诵）
- [概念 1]：[定义]
- [公式 1]：[公式内容]
- [理论 1]：[理论要点]

### 应用类（需要练习）
- [计算方法 1]：[步骤]
- [分析框架 1]：[框架]
```

**2.3 生成重点摘要**
```markdown
# {课程名} 核心知识点摘要

## 第 X 章：{章节名}
### 核心概念（必考）
1. **概念名**：一句话定义
2. **概念名**：一句话定义

### 重要公式
- 公式名：`表达式`

### 考点提示
- 选择题常考：[考点]
- 计算题常考：[考点]
- 简答题常考：[考点]
```

## 阶段 3：思维导图生成

**3.1 生成课程知识框架图**
```
调用：baoyu-diagram --type structural

生成要求：
- 中心主题：课程名称
- 一级分支：各章节
- 二级分支：核心概念/公式/方法
- 三级分支：详细要点
```

**3.2 生成章节概念图**
```
针对每章生成：
- 概念关系图（哪些概念相关）
- 流程图（计算步骤、分析流程）
- 对比图（相似概念对比）
```

**3.3 输出格式**
```
输出到：复习资料/{课程名}/思维导图/
- 课程总览图.svg
- 第 X 章概念图.svg
- 公式关系图.svg
```

## 阶段 4：记忆卡片制作

**4.1 Anki 卡片格式**
```markdown
卡片格式（正面 - 反面）：

正面：{问题/概念}
反面：{答案/解释}

示例：
正面：什么是会计等式？
反面：资产 = 负债 + 所有者权益

正面：流动比率的计算公式？
反面：流动比率 = 流动资产 ÷ 流动负债
```

**4.2 选择题卡片**
```markdown
正面：
{题目}
A. 选项 A
B. 选项 B
C. 选项 C
D. 选项 D

反面：
答案：{正确选项}
解析：{详细解释}
```

**4.3 批量生成卡片**
```
调用：自定义脚本

输入：知识点列表
输出：
- Anki 导入文件（.txt）
- Quizlet 导入文件（.csv）
- Markdown 卡片文档
```

**4.4 卡片分类**
```
按难度分级：
- 🔴 高难度（易错、常考）
- 🟡 中等难度（理解为主）
- 🟢 基础难度（记忆即可）

按题型分类：
- 选择题卡片
- 填空题卡片
- 简答题卡片
- 计算题卡片
```

## 阶段 5：习题整理

**5.1 习题提取**
```
从以下来源提取习题：
- 课后习题（PDF/Word）
- 作业题目
- 历年真题
- 模拟试卷
```

**5.2 错题本格式**
```markdown
# {课程名} 错题本

## 错题 1
**题目**：{题目内容}
**我的答案**：{错误答案}
**正确答案**：{正确答案}
**错误原因**：
- [ ] 概念不清
- [ ] 计算错误
- [ ] 审题失误
- [ ] 公式记错
**正确解法**：{详细步骤}
**相关知识点**：[知识点链接]()
**复习标记**：🔴 需重点复习
```

**5.3 题型分类整理**
```markdown
## 选择题专项
| 题号 | 知识点 | 难度 | 状态 |
|------|--------|------|------|
| 1 | 会计等式 | 🟢 | ✅ |
| 2 | 折旧计算 | 🟡 | ❌ |

## 计算题专项
| 题号 | 知识点 | 难度 | 状态 |
|------|--------|------|------|
| 1 | 财务报表分析 | 🔴 | ❌ |
| 2 | 成本核算 | 🟡 | ✅ |
```

## 阶段 6：复习计划制定

## 会计学特化功能

### 会计分录记忆卡片
```
正面：
计提固定资产折旧的会计分录？

反面：
借：制造费用/管理费用等
  贷：累计折旧
```

### 财务比率公式卡片
```
正面：
净资产收益率（ROE）的计算公式？

反面：
ROE = 净利润 ÷ 平均净资产
      = 净利率 × 总资产周转率 × 权益乘数
```

### 财务报表分析模板
```markdown
## 财务报表分析框架

### 1. 偿债能力分析
- 流动比率 = 流动资产 ÷ 流动负债
- 速动比率 = （流动资产 - 存货）÷ 流动负债
- 资产负债率 = 负债总额 ÷ 资产总额

### 2. 营运能力分析
- 应收账款周转率
- 存货周转率
- 总资产周转率

### 3. 盈利能力分析
- 销售毛利率
- 销售净利率
- 净资产收益率
```

## 使用示例

### 示例 1：期末考试复习

用户："帮我复习中级财务会计，下周考试"

工作流：
1. 读取课程笔记和 PPT
2. 提取核心知识点（资产、负债、所有者权益等）
3. 生成会计分录思维导图
4. 制作记忆卡片（会计等式、公式等）
5. 整理课后习题和作业
6. 制定 7 天复习计划

### 示例 2：制作记忆卡片

用户："帮我做会计学原理的记忆卡片"

工作流：
1. 读取知识库中的会计学资料
2. 提取需要记忆的概念和公式
3. 生成 Anki 格式的卡片
4. 按难度分级（基础、进阶、高频考点）
5. 输出可导入 Anki/Quizlet 的文件

## 注意事项

1. **提前规划**：至少提前 2 周开始系统复习
2. **理解优先**：不要死记硬背，先理解再记忆
3. **错题重做**：定期回顾错题本
4. **间隔重复**：使用 Anki 等工具进行间隔重复记忆
5. **模拟考试**：考前进行模拟测试

## 复习技巧

### 费曼学习法
1. 选择一个概念
2. 尝试用简单语言解释
3. 卡壳时回顾资料
4. 简化并建立类比

### 艾宾浩斯遗忘曲线
复习时间点：
- 学习后 1 小时
- 学习后 1 天
- 学习后 3 天
- 学习后 7 天
- 学习后 15 天

### 番茄工作法
- 25 分钟专注学习
- 5 分钟休息
- 4 个番茄后休息 15-30 分钟
