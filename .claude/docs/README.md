## 🎯 触发机制

**检测到关键词** → **计算置信度** → **决策执行**

| 置信度 | 决策 | 操作 |
|--------|------|------|
| ≥80% | 高置信 | 直接触发 skill |
| 60-79% | 中置信 | 触发 + 告知用户 |
| 40-59% | 低置信 | 询问用户确认 |
| <40% | 不触发 | 正常对话 |

**否定词不触发**：不想/不要/还没/别/没有
