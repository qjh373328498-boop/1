## 🐛 发现并修复的问题

### 问题 1：bc 命令未安装

**错误**:
```
./make_video.sh: line 373: bc: command not found
```

**修复**:
```bash
# 从 bc 改为 awk
frame_duration=$(awk "BEGIN {printf \"%.1f\", $audio_duration / $image_count}")
```

### 问题 2：FFmpeg filter_complex 语法错误

**错误**:
```
Stream specifier ':v' in filtergraph description matches no streams.
```

**原因**: 输入索引从 1 开始应为 0

**修复**:
```bash
# 改用 concat demuxer
ffmpeg -y -f concat -safe 0 -i concat_list.txt -i narration.mp3 ...
```
