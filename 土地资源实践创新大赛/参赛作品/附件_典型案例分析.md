# 典型案例分析（深度访谈）

> **第十届全国大学生土地资源实践创新大赛参赛作品附件**

---

## 案例一：积极型代表——土地资源管理专业大三学生

### 基本信息

| 项目 | 内容 |
|------|------|
| **编号** | I01 |
| **性别** | 男 |
| **年级** | 大三 |
| **专业** | 土地资源管理 |
| **生源地** | 农村（河南省周口市） |
| **认知得分** | 88 分 |
| **意愿类型** | 积极型 |
| **实际行为** | 参与过 3 次耕地保护活动 |

### 访谈摘录

**问：您是什么时候开始了解耕地保护的？**

> "从小在农村长大，家里种地，对耕地有感情。但真正系统了解是大学学了土地管理专业之后。大一的《土地资源学》课程第一次讲了 18 亿亩耕地红线，当时很震惊，没想到国家这么重视。"

**问：您主要通过哪些渠道获取相关信息？**

> "专业老师授课是最主要的。另外会关注自然资源部官网、中国土地科学微信公众号。偶尔看新闻也会留意相关政策。"

**问：您参与过哪些耕地保护行动？**

> "参加过三次：
> 1. 大二暑假跟老师去农村做耕地质量调研，入户调查 50 多户
> 2. 去年全国土地日参加学校组织的宣传活动，在社区发传单
> 3. 参与'田长制'志愿者项目，周末去郊区巡田"

**问：什么因素促使您参与这些活动？**

> "首先是专业相关，对以后就业有帮助。其次是觉得这件事确实有意义，农村耕地被占用的情况太普遍了，需要有人去做。还有就是老师和同学的推荐，身边人都在参与。"

**问：您认为阻碍其他同学参与的主要因素是什么？**

> "首先是不知道有什么渠道，很多同学想参与但找不到门路。其次是觉得个人作用有限，'我去了能改变什么'。还有就是时间冲突，周末有补习班或者兼职。"

**问：对学校耕地保护教育有什么建议？**

> "增加实践环节，不要只在课堂讲理论。可以组织更多实地考察，让学生亲眼看看耕地被破坏的情况，会有触动。另外可以邀请基层国土所的工作人员来讲课，讲实际工作中的案例。"

### 案例分析

**认知形成路径**：
```
农村成长经历 → 专业课程学习 → 教师引导 → 自主获取信息
```

**行为驱动因素**：
- 专业相关性（就业导向）
- 个人价值观认同（意义感）
- 同伴影响（从众效应）
- 制度激励（志愿时长、学分）

**典型特征**：
- 专业知识强，认知得分高
- 参与渠道畅通（老师推荐）
- 时间精力充足
- 目标明确（就业 + 公益）

## 案例二：被动型代表——汉语言文学专业大二学生

### 基本信息

| 项目 | 内容 |
|------|------|
| **编号** | I02 |
| **性别** | 女 |
| **年级** | 大二 |
| **专业** | 汉语言文学 |
| **生源地** | 城市（江苏省南京市） |
| **认知得分** | 65 分 |
| **意愿类型** | 被动型 |
| **实际行为** | 未参与过相关活动 |

### 访谈摘录

**问：您对耕地保护的了解主要来自哪里？**

> "高中地理课学过一些，大学就没有专门课程了。主要是刷抖音、微博的时候偶尔看到相关新闻，但不会仔细看。"

**问：您认为耕地保护重要吗？**

> "重要是肯定的，但感觉离我比较远。我是学文科的，以后应该不会接触这方面工作。而且城市里长大，对耕地没什么概念。"

**问：您是否愿意参与耕地保护活动？**

> "如果有合适的活动可以考虑，但不知道有什么可以参与的。学校好像没有组织过这方面的活动。如果是强制性的志愿服务可能会参加。"

**问：您认为什么因素会影响您的参与意愿？**

> "首先是要有趣，枯燥的活动不想参加。其次是时间不能太长，周末一天还可以接受。还有就是希望能有证书或者学分，对以后升学就业有帮助。"

**问：如果学校开设耕地保护通识课，您会选吗？**

> "看具体情况。如果学分好拿、作业不多，可能会选。如果很难就算了，专业课压力已经很大了。"

**问：您觉得什么样的宣传方式能吸引您？**

> "短视频吧，现在大家都是刷抖音、B 站。长篇大论的文章没人看。可以做一些有趣的科普动画，或者邀请网红博主来宣传。"

### 案例分析

**认知形成路径**：
```
高中地理课程 → 碎片化社交媒体 → 缺乏系统学习 → 认知停留在表面
```

**行为障碍因素**：
- 专业无关（缺乏动机）
- 城市成长（缺乏情感连接）
- 信息不对称（不知道参与渠道）
- 功利化倾向（需要证书/学分）

**典型特征**：
- 认知水平中等偏下
- 参与意愿被动
- 需要外部激励
- 偏好轻松有趣的形式

## 案例三：主动型代表——环境科学专业大四学生

### 基本信息

| 项目 | 内容 |
|------|------|
| **编号** | I03 |
| **性别** | 男 |
| **年级** | 大四 |
| **专业** | 环境科学 |
| **生源地** | 农村（湖南省娄底市） |
| **认知得分** | 82 分 |
| **意愿类型** | 主动型 |
| **实际行为** | 参与过 2 次活动，自发组织 1 次 |

### 访谈摘录

**问：您参与过哪些耕地保护活动？**

> "参加过学院组织的'送科技下乡'活动，去农村给农民讲土壤保护知识。另外自己组织过一次校园宣传活动，和几个同学一起，在食堂门口摆台，发传单、做问卷调查。"

**问：为什么会自发组织活动？**

> "做毕业论文需要收集数据，同时也觉得这是一个宣传的好机会。准备过程中发现很多同学对耕地保护的了解很有限，甚至不知道 18 亿亩红线。就想着做一些力所能及的事情。"

**问：活动效果怎么样？**

> "当天发了 300 多份传单，回收有效问卷 210 份。很多同学都停下来看了，也有人主动问问题。但感觉这种活动影响力有限，搞一天就结束了。"

**问：您认为如何提升活动影响力？**

> "需要常态化，不是一阵风。比如建立专门的社团，定期搞活动。还可以和媒体合作，扩大宣传。另外可以做一些线上活动，比如知识竞赛、打卡挑战，吸引更多同学参与。"

**问：您未来会继续参与耕地保护吗？**

> "会的。我申请了'三支一扶'，毕业后去基层工作，其中一项工作就是耕地保护。虽然现在力量小，但以后可以在工作岗位上发挥作用。"

### 案例分析

**认知形成路径**：
```
农村经历 → 专业学习 → 实践调研 → 主动传播
```

**行为特征**：
- 主动性强（自发组织）
- 目标明确（毕业论文 + 公益）
- 持续性好（毕业后继续）
- 影响力有限但有意改进

**典型特征**：
- 高认知 + 高意愿 + 高行为
- 专业相关 + 个人志向结合
- 有组织领导能力

## 案例四：冷漠型代表——计算机科学与技术专业大一学生

### 基本信息

| 项目 | 内容 |
|------|------|
| **编号** | I04 |
| **性别** | 男 |
| **年级** | 大一 |
| **专业** | 计算机科学与技术 |
| **生源地** | 城市（广东省深圳市） |
| **认知得分** | 52 分 |
| **意愿类型** | 冷漠型 |
| **实际行为** | 从未参与过 |

### 访谈摘录

**问：您了解耕地保护吗？**

> "听说过，但具体是什么不清楚。好像是说保护农田不能随便建房？"

**问：您认为这件事和您有关系吗？**

> "关系不大吧。我学计算机的，以后就是写代码、做开发。耕地保护应该是农业部门的事情。"

**问：如果学校组织相关活动，您会参加吗？**

> "可能不会。首先不感兴趣，其次时间有限，不如用来学习专业技能或者做项目。"

**问：什么情况下您可能会关注这件事？**

> "如果和就业有关可能会关注，比如参加'互联网 + 农业'的比赛。或者学校强制要求参加某些志愿活动。"

**问：您平时关注哪些社会议题？**

> "更多是科技发展、人工智能、互联网这些。耕地保护确实很少关注。"

**问：您认为如何提高像您这样的同学的参与度？**

> "可以把耕地保护和科技结合起来，比如用 AI 监测耕地变化，开发相关 APP。这样我们专业的人可能更有兴趣参与。"

### 案例分析

**认知状态**：
- 认知水平低（52 分）
- 认为与己无关
- 专业隔阂明显

**行为障碍**：
- 缺乏兴趣
- 时间机会成本高
- 缺乏结合点

**转化可能性**：
- 如果与专业结合（农业科技）有可能转化
- 需要强外部激励（就业、比赛）
- 短期内转化难度大

## 案例五：转变型代表——工商管理专业大三学生

### 基本信息

| 项目 | 内容 |
|------|------|
| **编号** | I05 |
| **性别** | 女 |
| **年级** | 大三 |
| **专业** | 工商管理 |
| **生源地** | 城市（四川省成都市） |
| **认知得分** | 75 分 |
| **意愿类型** | 主动型（经历过转变） |
| **实际行为** | 参与过 1 次活动 |

### 访谈摘录

**问：您对耕地保护的认识有变化吗？**

> "有比较大的变化。大一时觉得这件事和我无关，后来参加了一次社会实践活动，去农村调研土地流转，才发现问题比想象中严重。"

**问：是什么触发了您的转变？**

> "看到很多耕地被闲置或者用来建房子，农民说种地不赚钱。当时触动很大，觉得不能这样下去。回来后就主动学习相关知识，也参加了学校的宣传活动。"

**问：您认为什么因素促成了您的转变？**

> "首先是实地看到问题，有直观感受。其次是和农民交流，了解他们的困难。还有就是老师的引导，鼓励我们从专业角度思考解决方案。"

**问：您如何从专业角度参与耕地保护？**

> "我在想能不能用商业模式来解决。比如帮助农村发展乡村旅游、农产品电商，让农民有收入，就不会随便占用耕地了。现在在做一个创业计划，准备参加'互联网+'比赛。"

**问：对其他同学有什么建议？**

> "不要只在课堂学习，多出去看看。实地调研真的不一样，会有触动。另外可以结合专业找切入点，这样既有意义又对就业有帮助。"

### 案例分析

**转变路径**：
```
冷漠型 → 实地调研 → 情感触动 → 主动学习 → 主动参与
```

**关键转折点**：
- 实地调研（直观冲击）
- 农户访谈（情感连接）
- 教师引导（专业结合）
- 创业计划（持续动力）

**典型特征**：
- 经历过认知转变
- 能够结合专业
- 有持续参与动力

| **认知得分** | 85+ | 80+ | 60-70 | <60 | 70-80 |
| **参与次数** | 3+ | 2+ | 0-1 | 0 | 1-2 |
| **专业相关** | 强相关 | 相关 | 弱相关 | 无关 | 可结合 |
| **生源地** | 农村 | 农村/城市 | 城市 | 城市 | 城市 |
| **驱动因素** | 专业 + 价值 | 专业 + 志向 | 外部激励 | 基本无 | 情感触动 |
| **转化难度** | - | - | 中 | 高 | 已完成 |

### 认知形成机制

```
                    ┌─────────────┐
                    │  成长经历   │
                    │ (农村/城市) │
                    └──────┬──────┘
                           ↓
                    ┌─────────────┐
                    │  专业学习   │
                    │ (相关/无关) │
                    └──────┬──────┘
                           ↓
                    ┌─────────────┐
      ┌────────────│  信息获取   │────────────┐
      │            │ (系统/碎片) │            │
      │            └──────┬──────┘            │
      │                   ↓                   │
      │            ┌─────────────┐            │
      │            │  认知水平   │            │
      │            └──────┬──────┘            │
      │                   ↓                   │
      │            ┌─────────────┐            │
      │            │  态度评价   │            │

                主观规范、知觉行为控制、制度激励
```

### 关键洞察

1. **专业相关性是最强预测因子**
   - 相关专业（土地、环境）学生认知和行为显著高于非相关专业
   - 建议：加强非相关专业的通识教育

2. **农村成长经历有正向影响**
   - 农村生源对耕地有情感连接
   - 建议：城市学生增加实地体验

3. **实地调研能够促成认知转变**
   - 5 位受访者中 3 位提到实地经历的触动
   - 建议：增加实践环节

4. **专业结合是持续参与的关键**
   - 能够结合专业的学生参与度更高
   - 建议：设计跨学科参与项目

5. **外部激励对被动型有效**
   - 证书、学分、就业等激励能够提升参与意愿
   - 建议：完善激励机制
