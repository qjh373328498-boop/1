## 摘要

耕地保护关系国家粮食安全和经济社会可持续发展。大学生作为未来社会建设的中坚力量，其耕地保护认知水平和行为意愿直接影响耕地保护政策的实施效果。本研究采用问卷调查法，对全国 31 个省（自治区、直辖市）的 328 名在校大学生进行调查，运用描述性统计、Logistic 回归分析等方法，分析大学生耕地保护认知现状、行为意愿及其影响因素。研究结果表明：（1）大学生耕地保护认知整体处于中等水平，平均认知得分为 68.5 分；（2）约 72.3% 的大学生表示愿意参与耕地保护行动；（3）专业背景、生源地类型、政策关注度对耕地保护认知有显著正向影响；（4）认知水平、政策信任度、社会责任感对行为意愿有显著正向影响。基于研究结论，本文从加强耕地保护教育、创新宣传方式、完善参与机制等方面提出对策建议，为提升大学生耕地保护意识提供参考。

**关键词**：耕地保护；认知水平；行为意愿；大学生；Logistic 回归

## 1 调查概况

### 1.1 调查背景与意义

#### 1.1.1 政策背景

耕地是最宝贵的农业资源，是粮食生产的命根子。党的十八大以来，党中央高度重视耕地保护工作，提出"藏粮于地、藏粮于技"战略，实行最严格的耕地保护制度。2023 年中央一号文件明确提出"加强耕地保护和用途管控""严格耕地占补平衡管理"。2024 年，自然资源部会同相关部门印发《关于进一步加强耕地保护工作的通知》，要求全社会共同参与耕地保护。

然而，当前我国耕地保护形势依然严峻。据统计，2023 年全国耕地面积为 19.29 亿亩，人均耕地面积约 1.36 亩，不足世界平均水平的 40%。同时，耕地"非农化""非粮化"问题依然存在，耕地质量下降问题突出。在此背景下，提升全社会耕地保护意识，尤其是青年群体的认知水平，具有重要的现实意义。

#### 1.1.2 研究意义

大学生作为新时代的青年群体，是未来国家建设的主力军，其耕地保护认知水平和行为意愿直接影响耕地保护政策的长期效果。然而，现有研究多聚焦于农户、地方政府等主体，对大学生群体的关注相对不足。因此，开展大学生耕地保护认知与行为意愿调查，具有以下意义：

**理论意义**：丰富耕地保护公众参与理论研究，拓展认知 - 行为关系理论在土地资源管理领域的应用。

**实践意义**：为高校开展耕地保护教育提供参考，为政府部门制定耕地保护宣传政策提供依据。

---

### 1.2 调查目的与内容

#### 1.2.1 调查目的

1. 了解大学生对耕地保护政策、法规、知识的认知水平
2. 分析大学生参与耕地保护的行为意愿及其影响因素
3. 探究认知水平与行为意愿之间的关系
4. 提出提升大学生耕地保护意识和参与度的对策建议

#### 1.2.2 调查内容

| 调查模块 | 主要内容 |
|---------|---------|
| **基本信息** | 性别、年龄、年级、专业、生源地等 |
| **认知水平** | 耕地保护政策、法规、基本知识的了解程度 |
| **态度评价** | 对耕地保护重要性、紧迫性的认知 |
| **行为意愿** | 参与耕地保护宣传、监督、行动的意愿 |
| **影响因素** | 专业背景、信息来源、政策信任度等 |

#### 1.3.1 调查区域

本研究采用网络调查方式，调查范围覆盖全国 31 个省（自治区、直辖市）。根据经济发展水平和地理位置，将调查区域划分为东部、中部、西部三大板块。

### 1.4 调查方法与技术路线

#### 1.4.1 调查方法

**问卷调查法**：采用问卷星平台制作电子问卷，通过微信群、QQ 群、朋友圈等渠道发放，回收有效问卷 328 份。

**文献分析法**：通过知网、万方等数据库搜集耕地保护、公众参与、认知 - 行为理论等相关文献，为问卷设计和分析提供理论支撑。

**统计分析法**：运用 Excel 和 SPSS 软件进行描述性统计、信效度检验、Logistic 回归分析。

#### 1.4.2 技术路线

```
文献梳理 → 问卷设计 → 预调查 → 正式调查 → 数据整理 → 统计分析 → 结论建议
```

## 3 实证分析

### 3.1 研究假设

基于计划行为理论（TPB）和认知 - 行为理论，提出以下假设：

| 假设 | 内容 |
|------|------|
| H1 | 专业背景对耕地保护认知有显著正向影响 |
| H2 | 生源地类型对耕地保护认知有显著正向影响（农村>城市） |
| H3 | 政策关注度对耕地保护认知有显著正向影响 |
| H4 | 认知水平对行为意愿有显著正向影响 |
| H5 | 政策信任度对行为意愿有显著正向影响 |
| H6 | 社会责任感对行为意愿有显著正向影响 |

### 3.2 变量选取

### 3.3 模型构建

#### 3.3.1 线性回归模型（认知影响因素）

$$认知得分 = β₀ + β₁性别 + β₂年级 + β₃专业 + β₄生源地 + β₅政策关注 + ε$$

#### 3.3.2 Logistic 回归模型（行为意愿影响因素）

$$P(高意愿) = \frac{1}{1 + e^{-(β₀ + β₁认知 + β₂信任 + β₃责任感 + β₄关注 + ε)}}$$

### 3.4 回归结果分析

#### 3.4.1 认知水平影响因素分析（线性回归）

| 变量 | 系数 | 标准误 | t 值 | P 值 | VIF |
|------|------|--------|------|------|-----|
| 常数项 | 42.356 | 5.823 | 7.274 | 0.000*** | - |
| 性别 | -1.254 | 2.156 | -0.582 | 0.561 | 1.12 |
| 年级 | 2.385 | 0.856 | 2.786 | 0.006** | 1.25 |
| 专业 | 3.127 | 1.023 | 3.057 | 0.002** | 1.18 |
| 生源地 | 3.856 | 2.058 | 1.874 | 0.062* | 1.15 |
| 政策关注 | 5.236 | 1.125 | 4.654 | 0.000*** | 1.32 |

**模型拟合**：R² = 0.285，调整 R² = 0.272，F = 25.63（p<0.001）

**结果解读**：
- 年级每提升一级，认知得分增加 2.385 分（p<0.01）
- 经管类专业比文史类高 3.127 分（p<0.01）
- 政策关注度每提升 1 分，认知得分增加 5.236 分（p<0.001）
- 农村生源认知得分略高，但不显著（p=0.062）
- 性别影响不显著

**假设检验**：
- H1 专业背景→认知 ✓ 支持
- H2 生源地→认知 部分支持
- H3 政策关注→认知 ✓ 支持

| 性别 | -0.125 | 0.215 | 0.338 | 0.882 | 0.561 |
| 年级 | 0.156 | 0.098 | 2.532 | 1.169 | 0.112 |

**模型拟合**：-2LL = 285.63，Cox & Snell R² = 0.325，Nagelkerke R² = 0.412
**预测准确率**：78.5%

**结果解读**：
- 认知得分每提高 1 分，高意愿发生比增加 3.6%（p<0.001）
- 政策信任度每提升 1 分，高意愿发生比增加 69.6%（p<0.001）
- 社会责任感每提升 1 分，高意愿发生比增加 84.4%（p<0.001）
- 政策关注度对行为意愿有正向影响（p<0.05）

**假设检验**：
- H4 认知→行为意愿 ✓ 支持
- H5 政策信任→行为意愿 ✓ 支持
- H6 社会责任感→行为意愿 ✓ 支持

## 4 问题与原因

### 4.1 存在的主要问题

#### 4.1.1 认知水平整体偏低

- 平均得分仅 68.5 分（满分 100 分）
- 认知"优秀"群体仅占 8.5%
- 认知"较差"群体占 15.2%

**关键知识点薄弱环节**：
- 永久基本农田制度知晓率仅 45.1%
- 耕地保护补偿机制知晓率仅 48.5%
- 耕地质量保护措施知晓率仅 55.2%

#### 4.1.2 知行分离现象明显

| 认知水平 | 高意愿比例 | 低意愿比例 |
|---------|-----------|-----------|
| 优秀 | 92.9% | 7.1% |
| 良好 | 78.5% | 21.5% |
| 中等 | 70.4% | 29.6% |
| 及格 | 62.4% | 37.6% |
| 较差 | 48.0% | 52.0% |

即便认知"优秀"群体中，仍有 7.1% 意愿较低，存在"知而不行"现象。

#### 4.1.3 参与渠道匮乏

- 60.4% 的受访者表示"缺乏参与渠道"
- 47.6% 担心"被打击报复"
- 44.2% 认为"个人作用有限"

#### 4.1.4 信息获取渠道单一

| 渠道 | 使用率 | 有效性评价 |
|------|--------|-----------|
| 社交媒体 | 72.6% | 一般 |
| 新闻媒体 | 68.3% | 较好 |
| 学校课程 | 45.1% | 好 |
| 实地经历 | 32.3% | 很好 |

过度依赖碎片化信息，系统性知识获取不足。

---

### 4.2 问题产生的原因分析

#### 4.2.1 教育层面

1. **耕地保护教育缺失**
   - 高校课程设置中缺乏相关内容
   - 仅 45.1% 的学生通过课程了解耕地保护
   - 非相关专业几乎没有涉及

2. **教育形式单一**
   - 以课堂讲授为主，缺乏实践体验
   - 实地调研、案例教学不足
   - 学生参与度低

#### 4.2.2 宣传层面

1. **宣传内容专业化**
   - 政策文件语言晦涩难懂
   - 缺乏针对青年的宣传材料
   - 传播效果有限

2. **宣传渠道错位**
   - 官方宣传主渠道是政府网站
   - 大学生主渠道是社交媒体
   - 渠道错位导致信息触达率低

#### 4.2.3 制度层面

1. **公众参与机制不完善**
   - 缺乏便捷的参与渠道
   - 监督举报机制不健全
   - 激励机制缺失

2. **权益保障不足**
   - 举报人保护制度不完善
   - 47.6% 担心被打击报复
   - 影响参与积极性

#### 4.2.4 个人层面

1. **功利化倾向**
   - 更关注与就业、升学直接相关的知识
   - 耕地保护被视为"与己无关"
   - 9.8% 明确表示"与己无关"

2. **责任意识淡薄**
   - 社会责任感平均得分 4.12 分（有提升空间）
   - 个人行动与宏观问题关联认知不足

## 6 结论

### 6.1 主要结论

1. **认知水平中等**：大学生耕地保护认知平均得分为 68.5 分，处于中等水平。基础概念认知较好，具体政策制度认知不足。

2. **行为意愿较高**：72.3% 的大学生表示愿意参与耕地保护行动，综合意愿指数为 4.15（满分 5 分）。

3. **认知 - 意愿正相关**：认知水平与行为意愿呈显著正相关（r=0.426，p<0.001），认知水平越高，行为意愿越强。

4. **影响因素显著**：
   - 认知影响因素：年级、专业、政策关注度
   - 意愿影响因素：认知水平、政策信任度、社会责任感

5. **主要障碍**：缺乏参与渠道（60.4%）、担心被打击报复（47.6%）、认为个人作用有限（44.2%）。

### 6.2 创新点

1. **研究视角创新**：聚焦大学生群体，填补耕地保护公众参与研究的空白。

2. **理论框架创新**：整合计划行为理论和认知 - 行为理论，构建"认知 - 态度 - 意愿"分析框架。

3. **方法应用创新**：运用 Logistic 回归和中介效应检验，深入分析影响因素和作用机制。

### 6.3 研究局限

1. **样本局限**：样本以在校生为主，毕业生群体覆盖不足。

2. **方法局限**：横截面数据无法捕捉认知和意愿的动态变化。

3. **测量局限**：行为意愿与实际行为可能存在差距，需要追踪研究。

4. **区域局限**：网络调查可能导致区域样本分布不均。
