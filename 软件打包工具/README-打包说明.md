# 📦 软件打包完整方案

## 🎯 目标
创建用户可**直接运行**的 Windows 安装包，无需安装 Python 或配置环境。

### 使用场景
- 快速分发给用户
- 无需安装，解压即用
- 可放在 U 盘随身携带

## 📋 打包步骤

### 方式 A：自动化脚本（推荐）

1. **在 Windows 电脑上**打开 `软件打包工具/` 文件夹

2. **双击运行** `一键打包-Windows.bat`

3. 等待自动完成：
   - ✅ 检查 Python 环境
   - ✅ 安装 PyInstaller 和依赖
   - ✅ 打包学创杯辅助软件
   - ✅ 打包财务工具箱
   - ✅ 创建启动菜单

4. 完成后获得：
   - `学创杯辅助软件/dist/学创杯辅助软件.exe`
   - `财务工具箱/dist/财务工具箱.exe`

5. 复制到 `软件/` 文件夹，与 `启动菜单.bat` 放在一起

6. 压缩整个 `软件/` 文件夹，分发给用户

### 方式 B：手动打包（高级）

详见 **《Windows 打包指南.md》**

## 🎁 用户如何使用

### 1. 下载压缩包
用户收到 `软件.zip` 或 `软件.rar`

### 2. 解压到任意位置
例如：`C:\Users\用户名\Desktop\软件\`

### 3. 双击 `启动菜单.bat`
选择要启动的软件即可

### 4. 浏览器自动打开
默认访问：
- 学创杯辅助软件：`http://localhost:8502`
- 财务工具箱：`http://localhost:8501`（密码：703102）

## ⚠️ 注意事项

### 1. 文件大小
- 每个 .exe 约 15-20MB（包含完整 Python 环境）
- 总大小约 40-50MB

### 2. 首次启动慢
- 首次运行需解压内置 Python 环境（10-30 秒）
- 后续启动速度正常

### 3. 杀毒软件
- Windows Defender 可能提示"未知发布者"
- 点击"仍要运行"即可
- 某些杀毒软件可能误报，需添加信任

### 4. 系统要求
- Windows 10/11
- 4GB+ 内存
- 无需安装 Python

### 5. 端口占用
确保 8501 和 8502 端口未被占用

## 🔧 打包优化技巧

### 1. 自定义图标

准备 `.ico` 文件，打包时添加：
```batch
pyinstaller ... --icon=myapp.ico app.py
```

### 2. 减小体积

使用 UPX 压缩（需先安装 UPX）：
```batch
pyinstaller --upx-dir=upx ... app.py
```

### 3. 版本信息

创建 `version.txt` 添加版本信息：
```batch
pyinstaller ... --version-file=version.txt app.py
```

### 4. 多用户测试

打包后在多台电脑测试，确保兼容性

## 🆘 常见问题

**Q: 打包后运行提示"找不到模块"？**
A: 确保添加了所有 `--hidden-import` 参数

**Q: 杀毒软件报毒怎么办？**
A: 这是 PyInstaller 打包的常见问题，添加到杀毒软件白名单

**Q: 如何在 Mac 上打包？**
A: 需要在 Mac 上使用 PyInstaller，打包出的是 .app 文件

**Q: 可以同时启动两个软件吗？**
A: 可以，启动菜单选择"同时启动两个软件"，或在两个浏览器窗口分别访问不同端口
