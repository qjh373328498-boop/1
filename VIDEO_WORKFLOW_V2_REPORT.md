## ✅ V2.0 改进内容

### 改进 1：智能脚本生成器

**新增文件**: `.claude/scripts/gen_script_smart.sh`

**功能**: 根据视频时长自动生成有内容的脚本（非模板）

**支持三种结构**:

#### 短视频（≤60 秒）
- 开场 (10s)
- 3 个核心要点 (35s)
- 总结 (15s)

#### 中等视频（60-180 秒）
- 开场 (20s)
- 第一部分：概念介绍 (40s)
- 第二部分：重要性 (40s)
- 第三部分：应用方法 (40s)4
- 结尾 (40s)

#### 长视频（>180 秒）
- 开场 (30s)
- 第一章：基础概念 (90s)
- 第二章：重要性 (90s)
- 第三章：方法论 (90s)
- 第四章：实战案例 (90s)
- 总结与建议 (60s)

**测试示例**:
```bash
./gen_script_smart.sh "德国心脏病桌游" 180 script.md
```

**输出**:
- 1790 字符
- 结构完整
- 有实际内容（非占位符）

### 改进 2：动态分镜生成

**修改**: `step3_generate_storyboard()`

**优化前**:
```bash
# 固定 5 场景
scenes = [
  {"id": 1, "time": "0:00-0:20", ...},
  {"id": 2, "time": "0:20-1:00", ...},
  {"id": 3, "time": "1:00-1:40", ...},
  {"id": 4, "time": "1:40-2:20", ...},
  {"id": 5, "time": "2:20-3:00", ...}
]
```

**优化后**:
```python
# 动态分析脚本章节
sections = re.findall(r'^## (.+)$', content, re.MULTILINE)

scenes = []
for i, section in enumerate(sections, 1):
    scenes.append({
        "id": i,
        "title": section,
        "image_prompt": f"与\"{section}\"相关的插图...",
    })
```

**效果**:
- 根据脚本章节数量动态生成场景
- 场景数不固定（3-8 个）
- 每个场景对应脚本内容

## 🔧 技术改进

### 1. 智能脚本生成算法

```bash
# 根据时长自动选择结构
if [ "$DURATION" -le 60 ]; then
    # 短视频结构
    generate_short_script
elif [ "$DURATION" -le 180 ]; then
    # 中等视频结构
    generate_medium_script
else
    # 长视频结构
    generate_long_script
fi
```

### 2. 动态分镜提取

```python
# 使用正则表达式提取章节
sections = re.findall(r'^## (.+)$', content, re.MULTILINE)

# 动态生成场景
for i, section in enumerate(sections, 1):
    scenes.append({
        "id": i,
        "title": section,
        "image_prompt": f"与\"{section}\"相关的插图...",
        "narration": f"关于{section}的内容"
    })
```

### 3. 容错机制

```python
try:
    # 尝试动态生成
    sections = extract_sections(script_file)
    scenes = generate_scenes(sections)
except Exception as e:
    # 降级到默认模板
    print(f"⚠️  分镜生成失败，使用默认模板：{e}")
    scenes = DEFAULT_SCENES
```
