# 🚀 软件启动工具 - 使用说明

## 📁 文件夹内容

| 文件名 | 适用系统 | 说明 |
|--------|---------|------|
| `一键启动-Windows 系统.bat` | Windows 10/11 | 双击运行 |
| `一键启动-Mac-Linux 系统.sh` | macOS / Linux | 需要执行权限 |
| `使用说明.md` | 所有系统 | 本文档 |

---

## 💻 Windows 系统

### 启动方法
**双击运行**: `一键启动-Windows 系统.bat`

### 首次运行
如果提示缺少 Python：
1. 访问 https://www.python.org/downloads/
2. 下载并安装 Python 3.8+
3. **重要**: 安装时勾选 "Add Python to PATH"

或使用 winget 一键安装：
```
winget install Python.Python.3.11
```

---

## 🍎 macOS 系统

### 启动方法
```bash
# 1. 打开终端 (Terminal)
# 2. 进入本文件夹
cd /path/to/软件启动工具

# 3. 添加执行权限
chmod +x 一键启动-Mac-Linux 系统.sh

# 4. 运行
./一键启动-Mac-Linux 系统.sh
```

### 首次运行
如果提示缺少 Python：
```bash
brew install python3
```

---

## 🐧 Linux 系统

### 启动方法
```bash
# 1. 进入本文件夹
cd /path/to/软件启动工具

# 2. 添加执行权限
chmod +x 一键启动-Mac-Linux 系统.sh

# 3. 运行
./一键启动-Mac-Linux 系统.sh
```

### Ubuntu/Debian
如果提示缺少 Python：
```bash
sudo apt update
sudo apt install python3 python3-pip
```

### CentOS/RHEL
```bash
sudo yum install python3 python3-pip
```

## ⚠️ 注意事项

1. **首次运行** 会自动安装 Streamlit 和依赖，约 1-2 分钟
2. **端口占用** 如提示端口被占用，关闭其他程序
3. **防火墙** 如无法访问，检查防火墙设置
4. **Python 版本** 需要 Python 3.8 或更高

## 🔧 常见问题

### Q: 双击没反应？
A: Windows 可能隐藏了文件扩展名，确认文件是 `.bat` 结尾

### Q: 提示"permission denied"？
A: Mac/Linux 需要先执行 `chmod +x 一键启动-Mac-Linux 系统.sh`

### Q: 浏览器没打开？
A: 手动访问 http://localhost:8501 或 http://localhost:8502

### Q: Streamlit 安装失败？
A: 选择菜单选项 4 重新安装所有依赖

### Q: 下载后如何使用？
A: 整个 `软件启动工具` 文件夹复制到任何位置，双击即可运行
