# 💾 财务工具箱 - 本地安装指南

**更新日期**: 2026-05-03

---

## ⚠️ 重要提示

### 路径要求

由于 Windows 中文路径兼容性问题，请**务必**将软件放在**纯英文路径**下：

✅ **推荐路径**:
```
D:\Software\財務工具箱\
C:\Apps\CW_Toolbox\
E:\tools\finance\
```

❌ **避免路径**:
```
G:\软件\财务工具箱\     (中文目录)
G:\??2\??\             (乱码路径)
C:\Users\张三\软件\     (用户名含中文)
```

## 📥 安装步骤

### 方法 1：使用启动器（推荐）

1. **下载软件到合适位置**
   - 建议放在：`D:\Software\` 或 `C:\Apps\`
   - 路径中不要有中文

2. **运行一键启动**
   ```
   双击：一键启动.bat
   ```

3. **选择财务工具箱**
   ```
   请输入选项 (0-3): 2
   ```

4. **等待自动安装**
   - 首次运行会自动创建虚拟环境
   - 自动安装依赖（约 2-3 分钟）
   - 完成后自动打开浏览器

### 方法 2：手动安装

1. **打开命令提示符**
   ```
   Win + R → 输入 cmd → 回车
   ```

2. **进入软件目录**
   ```cmd
   cd /d D:\Software\财务工具箱
   ```

3. **创建虚拟环境**
   ```cmd
   python -m venv venv
   ```

4. **激活虚拟环境**
   ```cmd
   venv\Scripts\activate
   ```

5. **安装依赖**
   ```cmd
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

6. **启动软件**
   ```cmd
   python -m streamlit run app.py
   ```

## 🔧 常见问题

### Q1: 路径有中文，出现乱码怎么办？

**解决**: 将软件移动到纯英文路径下：
```
G:\软件\财务工具箱\  →  D:\Software\CW_Toolbox\
```

### Q2: 提示"找不到 Python"

**原因**: 未安装 Python 或不在 PATH 中

**解决**:
1. 下载 Python 3.8+: https://www.python.org/downloads/
2. 安装时勾选 ✅ "Add Python to PATH"
3. 重新打开 CMD 测试：`python --version`

### Q3: pip 安装失败

**解决**:
```cmd
# 升级 pip
python -m pip install --upgrade pip

# 手动安装依赖
pip install streamlit pandas plotly openpyxl -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q4: 依赖安装成功但启动失败

**解决**:
```cmd
# 删除虚拟环境重新创建
rmdir /s /q venv
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m streamlit run app.py
```

### Q5: 浏览器不自动打开

**解决**: 手动访问 http://localhost:8501

## 🎯 优化建议

### 1. 第一次启动慢

**正常现象**，首次运行需要：
- 创建虚拟环境（约 30 秒）
- 下载安装依赖（约 1-2 分钟）
- 第二次启动会快很多

### 2. 加速下载

已配置清华镜像源，如果仍然慢：
```cmd
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

### 3. 定期清理

建议每月清理一次缓存：
```cmd
# 删除 venv 重新创建
rmdir /s /q venv
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```
