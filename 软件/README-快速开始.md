# 🚀 学创杯 + 财务工具箱 - 快速开始

## 📦 下载后直接使用

**无需安装，解压即可运行！**

---

## 💻 Windows 用户

### 第 1 步：解压文件
将下载的压缩包解压到任意位置，例如：
- `C:/学创杯工具/`
- `D:/比赛软件/`
- 桌面

### 第 2 步：一键启动
双击运行：
```
软件/软件启动工具/一键启动-Windows 系统.bat
```

### 第 3 步：选择软件
```
请选择要启动的软件：

  1) 🏆 学创杯辅助工具 v2.2 (端口 8502)
  2) 💰 财务工具箱 v1.5 (端口 8501)
  3) 📊 同时启动两个软件
  4) ❌ 退出
```

### 第 4 步：浏览器访问
自动打开浏览器，或手动访问：
- 学创杯：http://localhost:8502
- 财务工具箱：http://localhost:8501 (admin / 703102)

## 🍎 Mac / 🐧 Linux 用户

### 第 1 步：解压文件
```bash
unzip 软件.zip -d ~/学创杯工具/
cd ~/学创杯工具/软件
```

### 第 2 步：添加执行权限
```bash
chmod +x 软件启动工具/一键启动-Mac-Linux 系统.sh
```

### 第 3 步：运行启动脚本
```bash
./软件启动工具/一键启动-Mac-Linux 系统.sh
```

### 第 4 步：选择软件
输入数字 `1` 或 `2` 选择要启动的软件

## ⚠️ 首次运行说明

### 自动安装依赖
首次运行时会自动安装 Python 依赖，约 1-2 分钟

### 如果提示缺少 Python
**Windows 用户**:
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.8+
3. **重要**: 安装时勾选 "Add Python to PATH"

**Mac 用户**:
```bash
brew install python3
```

**Linux 用户**:
```bash
sudo apt install python3 python3-pip  # Ubuntu/Debian
sudo yum install python3 python3-pip  # CentOS
```

## 🔧 故障排除

### 问题 1: 双击没反应
- Windows 可能隐藏了文件扩展名
- 确认文件是 `.bat` 或 `.sh` 结尾

### 问题 2: 提示"permission denied"
- Mac/Linux 需要执行权限
- `chmod +x 软件启动工具/一键启动-Mac-Linux 系统.sh`

### 问题 3: 浏览器没打开
- 手动访问 http://localhost:8501 或 8502

### 问题 4: 端口被占用
- 关闭其他程序
- 或修改脚本中的端口号
