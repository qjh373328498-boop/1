# 📦 打包成 EXE 指南

## 🎯 快速开始

本项目已准备好**一键打包脚本**，在 Windows 电脑上双击即可生成 `.exe` 安装包。

---

## 🚀 三步打包

### 步骤 1️⃣：准备 Windows 电脑

确保电脑已安装 **Python 3.8+**

检查方法：
```cmd
python --version
```

如未安装，请前往 https://www.python.org/downloads/ 下载安装

### 步骤 2️⃣：运行打包脚本

1. 打开 **`软件打包工具/`** 文件夹
2. 双击运行 **`一键打包-Windows.bat`**
3. 等待自动完成（约 5-10 分钟）

### 步骤 3️⃣：获取打包文件

完成后在以下位置找到 `.exe` 文件：
- `学创杯辅助软件/dist/学创杯辅助软件.exe`
- `财务工具箱/dist/财务工具箱.exe`

---

## 📁 分发给用户

### 方法 1：绿色版（推荐）

将以下文件复制到一起：
```
软件/
├── 启动菜单.bat
├── 学创杯辅助软件.exe
├── 财务工具箱.exe
└── 使用说明.md
```

压缩为 `软件.zip`，用户解压后双击 `启动菜单.bat` 即可使用

### 方法 2：单独分发

直接将两个 `.exe` 文件发送给用户，用户双击即可运行

---

## ⚠️ 注意事项

1. **首次启动较慢**（10-30 秒，解压内置 Python 环境）
2. **杀毒软件可能误报**（PyInstaller 打包常见，可添加信任）
3. **每个 exe 约 15-20MB**（包含完整 Python 环境）
